# [Бэкенд конфиг тестов (pytest.ini)](../../../Backend/PlayerData/pytest.ini)

## Назначение
Локальный файл конфигурации `pytest` для папки `PlayerData`. 

## Настройки
*   **`-s`**: Разрешает вывод `print` в консоль во время тестов.
*   **`-ra`**: Показывает краткий отчет по завершении.
*   **`-W ignore`**: Игнорирует системные предупреждения Python, чтобы не засорять логи.
