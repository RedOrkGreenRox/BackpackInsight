# [Сэмплы билдов и логов (PlayerData/builds/)](../../../../Backend/PlayerData/builds/)

## Назначение
Папка содержит дополнительные примеры игровых состояний, используемые для отладки конкретных функций отображения билдов.

## Содержимое
*   **German_build_1.json**: Снимок финального билда игрока (расположение предметов в сумке).
*   **German_log_1.json**: Лог боя, используемый для тестирования парсера событий.
*   **Sky_Dorf_1.json**: Пример данных для героя Dorf.
