# [Сырая выгрузка предметов (itemdefinition_*.csv)](../../../../Backend/PlayerData/builds/)

## Назначение
Файлы формата `itemdefinition_YYYYMMDDHHMM.csv` представляют собой сырые экспорты данных из Unity-проекта игры Backpack Brawl.

## Роль в проекте
Эти файлы служат промежуточным звеном при обновлении базы знаний. Они содержат максимально подробную техническую информацию, которая затем конвертируется в [items_*.json](../../db/items_versions.md) для использования в API.

## Структура
CSV включает технические колонки:
*   Внутренние ID объектов.
*   Ссылки на префабы и ассеты.
*   Базовые параметры баланса до их обработки парсером.

## AI-контекст
*   Этот файл не используется приложением в рантайме. Он необходим разработчикам для контроля изменений в балансе игры на низком уровне.
