# [Сборка-образец German_build_1 (German_build_1.json)](../../../../Backend/PlayerData/builds/German_build_1.json)

## Назначение
Пример сохранённой «сборки» (рюкзак с предметами и их позициями) игрока German. Тестовые/демо-данные.

## Структура
*   `inventoryItems[]`: `{id, name, position{x,y}, orientation, slotPositions[]}` — предметы и их раскладка в рюкзаке.

## Связи (Dependencies)
*   Формат сборок обрабатывается утилитами [builds/icon_parser.py](icon_parser.md) и связан с парсером иконок предметов.

## AI-контекст
*   Образец данных, не рантайм. Полезен для тестов отображения раскладки рюкзака.
