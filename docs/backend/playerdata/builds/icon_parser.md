# [Синхронизированный парсер иконок (icon_parser.py)](../../../../Backend/PlayerData/builds/icon_parser.py)

## Назначение
Модуль `icon_parser.py` — это высокоуровневый аналитический инструмент, полностью синхронизированный с фронтенд-версией (`icon-parser.ts`). Он обеспечивает идентичное отображение игровых текстов на сервере и клиенте, а также подготавливает данные для симуляции боя.

---

## Ключевая функциональность

### 1. Визуальная синхронизация (Identical HTML)
Модуль реализует многоэтапный процесс преобразования сырого текста в HTML:
*   **Детекция иконок**: Распознает более 80 типов игровых сущностей (от `Damage` до специфических героев типа `Kragg`).
*   **Умные заголовки**: Выделяет логические блоки в описаниях, используя сложную систему исключений (напр. игнорирование `Cost`, `Hero`).
*   **Форматирование значений**: Автоматически оборачивает числа и проценты в `<span class="value-text">` для стилизации.
*   **Специальная логика**: Поддержка уникальных правил для иконок типа `Star` (Dragonleaf logic), где иконка влияет на окраску последующих слов.

### 2. Структурированный анализ (Combat Simulation)
В отличие от фронтенд-версии, Python-модуль дополнительно извлекает машиночитаемую структуру:
*   **Triggers**: События активации (`On hit`, `Every X seconds`).
*   **Passives**: Постоянные эффекты.
*   **Effect Details**: Декомпозиция строк на конкретные действия (`Gain`, `Inflict`), числовые значения и типы статов.

### 3. Поддержка современных форматов
Генерирует HTML-теги `<picture>` с фоллбэками:
1.  **AVIF**: Максимальное сжатие.
2.  **WebP**: Оптимальный баланс.
3.  **PNG/Default**: Для старых систем.
