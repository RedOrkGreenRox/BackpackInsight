# [Пакет Моделей (__init__.py)](../../../../Backend/PlayerData/models/__init__.py)

## Назначение
Этот файл экспортирует основные модели проекта (`Profile`, `Item`, `ItemDefinition`, `Hero`).

## Роль в системе
Помимо удобства импортов, этот файл критически важен для **регистрации метаданных**. При импорте моделей в [env.py](../../db/migrations/env.md) Alembic видит все таблицы проекта и может генерировать автоматические миграции.
