# [Dockerfile Бэкенда (Dockerfile)](../../../Backend/PlayerData/Dockerfile.md)

## Назначение
Инструкции для сборки Docker-образа бэкенд-сервиса.

## Процесс сборки
1.  **База**: `python:3.11-slim` (минималистичный образ).
2.  **Среда**: Установка системных зависимостей для работы PostgreSQL (`libpq-dev`, `gcc`).
3.  **Код**: Копирование всего содержимого репозитория в `/app`.
4.  **Команда запуска**:
    ```bash
    sh -c "python -m Backend.DB.bootstrap && uvicorn Backend.PlayerData.api:app --host 0.0.0.0 --port 8000"
    ```
    *   Сначала запускается [bootstrap.py](../../db/bootstrap.md) для подготовки БД.
    *   Затем стартует основной сервер на порту 8000.
