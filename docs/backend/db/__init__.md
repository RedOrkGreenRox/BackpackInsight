# [Пакет Базы Данных (__init__.py)](../../../Backend/DB/__init__.py)

## Назначение
Файл `Backend/DB/__init__.py` делает директорию `DB` полноценным Python-пакетом.

## Роль
Он пуст, но его наличие необходимо для того, чтобы другие части приложения могли импортировать модули базы данных, например:
```python
from Backend.DB.database import get_session
```
