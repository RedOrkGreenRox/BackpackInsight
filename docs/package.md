# [Корневой package.json (package.json)](../package.json)

## Назначение
Этот файл является техническим манифестом в корне проекта. Он не относится напрямую к логике работы фронтенда или бэкенда, а служит для управления **инструментами разработки всего репозитория**.

## Содержимое
В текущей версии содержит единственную зависимость:
*   **`repomix`**: Специализированный инструмент, который упаковывает всю структуру репозитория и кодовую базу в один текстовый файл. Это крайне удобно для передачи контекста проекта в большие языковые модели (LLM) для глубокого анализа или генерации документации.

## AI-контекст
*   **Разделение ответственности**: Не путать этот файл с [Frontend/Web/package.json](Frontend/package.md). Все зависимости веб-приложения (Vite, TypeScript, Sass) находятся внутри папки `Frontend/`.
*   Корневой `package.json` позволяет запускать инструменты уровня проекта (например, линтеры документации или упаковщики кода) одной командой из корня.
