# [Интеграционные тесты Профилей (test_profiles_integration.py)](../../tests/test_profiles_integration.py)

## Назначение
Масштабный интеграционный тест, который проверяет парсинг **реальных** JSON-логов из папки `Backend/PlayerData/Profiles/`.

## Логика теста
*   Скрипт автоматически находит все файлы `.json` в директории профилей.
*   Использует `pytest.mark.parametrize` для запуска отдельного теста на каждый файл.
*   Проверяет целостность данных после сохранения в БД: отсутствие неизвестных героев и предметов.
*   **Контроль XP**: Сравнивает суммарный опыт, рассчитанный на основе инвентаря, со значением опыта, указанным в профиле. Выводит предупреждения при расхождениях.

---

## AI-контекст
*   Этот тест является «лакмусовой бумажкой» при обновлении игры. Если разработчики игры меняют формат лога, этот тест упадет первым, указав на конкретные файлы с ошибками.
