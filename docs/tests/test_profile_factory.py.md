# [Тесты Фабрики (test_profile_factory.py)](../../tests/test_profile_factory.py)

## Назначение
Детальные тесты сервиса [ProfileFactory](../backend/playerdata/services/ProfileFactory.md), отвечающего за парсинг и кэширование.

## Что тестируется
1.  **Валидация**: Реакция на отсутствие ключей `Data`, `Name`, `UID` в логах.
2.  **Синхронизация кэша**: Проверка наполнения кэша определений предметов при старте.
3.  **Статистика кэша**: Корректность подсчета попаданий/промахов (Hit/Miss Rate).
4.  **Парсинг косметики**: Тестирование регулярных выражений для извлечения скинов и баннеров из длинного списка разблокировок (`UL`).
5.  **Интеграция**: Сборка полного объекта профиля из сырого JSON с проверкой курсов валют и трофеев.
