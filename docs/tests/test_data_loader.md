# [Тесты загрузчика данных (test_data_loader.py)](../../tests/test_data_loader.py)

## Назначение
Тестирование функций загрузки статических справочников из файлов `items_*.json`.

## Ключевые проверки
1.  **Объем данных**: Проверка того, что загружается существенное количество предметов (>100), что исключает загрузку пустых файлов.
2.  **Поиск предметов**: Проверка наличия ключевых предметов (Apple, Wooden Sword) в словаре.
3.  **Авто-версионирование**: Тестирование функции `get_latest_items_file`, которая должна находить файл с максимальным номером версии в имени.
