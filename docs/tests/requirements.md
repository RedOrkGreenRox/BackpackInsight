# [Зависимости тестов (requirements.txt)](../../tests/requirements.txt)

## Назначение
Зависимости для запуска тестов: фреймворк + дублирование зависимостей приложения (чтобы импортировать его модули).

## Ключевое
*   `pytest`, `pytest-cov`, `httpx`.
*   Дублируют рантайм: `fastapi`, `sqlmodel`, `pydantic`, `python-dotenv`, `alembic`, `slowapi`.

## AI-контекст
*   Тесты гоняют под SQLite (`USE_SQLITE=True`), но требуют тех же пакетов, что и приложение.
