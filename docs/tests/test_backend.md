## Назначение
Общие интеграционные тесты для проверки работоспособности бэкенда.

## Ключевые проверки
1.  **Загрузка данных**: Подтверждение того, что статические предметы успешно импортированы в базу данных при инициализации сессии.
2.  **Создание профиля**: Сквозная проверка цепочки `JSON -> Profile -> DB -> Refresh`. Убеждается, что после сохранения в базу все поля профиля и связанных предметов доступны.
