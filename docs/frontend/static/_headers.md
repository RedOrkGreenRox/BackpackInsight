# [HTTP-заголовки (копия в static) (_headers)](../../../Frontend/Web/static/_headers)

## Назначение
Копия правил кэширования в каталоге `static/`, чтобы файл попал в собранную раздачу `dist/` (некоторые хостинги ждут `_headers` в корне публикации).

## Связи (Dependencies)
*   Содержательно повторяет [../_headers](../_headers.md). Семантика правил описана там.

## AI-контекст
*   Держите обе копии синхронными: при изменении политик кэша правьте и `Frontend/Web/_headers`, и `Frontend/Web/static/_headers`.
