# [Синонимы для поиска (term-aliases.ru.json)](../../../../Frontend/Web/static/search/term-aliases.ru.json)

## Назначение
База синонимов/переводов игровых терминов для семантического поиска: канонический термин (en) → список русских синонимов. Позволяет находить предметы по запросу «яд», «крит», «броня» и т.п.

## Связи (Dependencies)
*   Загружается [SearchTermService](../../ground/utils/SearchTermService.md) для расширения запросов.
*   Кандидаты в синонимы помогает выявлять [analyze-item-text.js](../../scripts/analyze-item-text.md).

## Структура
*   Объект: `"damage": ["урон","дамаг",...]`, `"critical": [...]`, `"stamina": [...]` и десятки других терминов (статы, типы оружия, механики).

## AI-контекст
*   Ядро качества русского поиска. При добавлении новой игровой механики/стата добавляйте сюда синонимы. Ключи — канонические англ. термины, совпадающие с логикой [SearchTermService](../../ground/utils/SearchTermService.md).
