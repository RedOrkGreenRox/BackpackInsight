# [Русские алиасы поисковых терминов (term-aliases.ru.json)](../../../../Frontend/Web/static/search/term-aliases.ru.json)

## Назначение
`term-aliases.ru.json` — словарь канонических игровых терминов и русских пользовательских формулировок для семантического поиска страницы предметов.

## Текущая роль
Словарь используется в двух местах:
- [SearchTermService](../../ground/utils/SearchTermService.md) расширяет `searchText` предметов.
- [alias-fuzzy](../../ground/branches/items/_items/managers/filter/alias-fuzzy.md) строит Fuse-индекс по самим алиасам, чтобы запросы с небольшими ошибками (`пчола`) находили ближайший словарный термин (`пчела -> bee`).

## Принципы наполнения
- В словарь добавляются адекватные слова и распространённые формы, а не бесконечный список опечаток.
- Опечатки должен покрывать fuzzy-слой `alias-fuzzy.ts`.
- Короткие слова требуют осторожности: агрессивный fuzzy по ним может давать шум.
