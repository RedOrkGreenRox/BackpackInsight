# 🖼 Иконки установки PWA (images/manifest/)

## Назначение
Иконки приложения для установки как PWA: размеры 72…512, в **трёх** форматах — AVIF, WebP и PNG (PNG нужен для максимальной совместимости установщиков).

## Структура
*   `avif/icon-<size>.avif`, `webp/icon-<size>.webp`, `png/icon-<size>.png` (+ `logo.png`).

## Связи (Dependencies)
*   Перечислены в [manifest.json](../../manifest.md) (поле `icons`) и apple-touch-icon в [index.html](../../../index.html.md).

## AI-контекст
*   Единственная папка с PNG — намеренно (не все платформы понимают AVIF/WebP для иконок установки). Версионируются через `?v=5` в manifest.
