# [Локализация: русский (ru.json)](../../../../Frontend/Web/static/lang/ru.json)

## Назначение
Словарь переводов интерфейса на русском: ключ → строка. Загружается системой i18n при старте/смене языка.

## Связи (Dependencies)
*   Читается [i18n](../../ground/localization/i18n.md) (функция `t(key)`).
*   Прекэшируется [sw.js](../sw.md); кэш-политика — [_headers](../_headers.md) (1 час).

## Структура
*   Плоский объект ключ→значение: `sidebar_*`, `profile_*`, `error_*`, `item_*`, `not_found_*` и т.д. Поддерживает параметры-подстановки (например, `item_detail_title`).

## AI-контекст
*   Должен быть согласован по набору ключей с [en.json](en.md). Добавляя ключ в код через `t('...')`, добавьте его в ОБА словаря, иначе на одном языке будет «дыра».
