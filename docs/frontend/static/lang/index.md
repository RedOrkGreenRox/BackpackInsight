# [Словари локализации (lang/*.json)](../../../../Frontend/Web/static/lang/)

## Назначение
Папка `lang/` содержит JSON-файлы с переводами интерфейса на разные языки. Это основной источник данных для модуля [i18n.ts](../../ground/localization/i18n.md).

---

## Файлы
*   **`ru.json`**: Полный перевод на русский язык (основной).
*   **`en.json`**: Перевод на английский язык.

## Структура Ключей
Тексты организованы по функциональным блокам:
*   `sidebar_*`: Навигационное меню.
*   `profile_*`: Страница аналитики профиля.
*   `items_*`: Библиотека и фильтры предметов.
*   `error_*`: Системные уведомления и ошибки.
*   `json_*`: Ошибки валидатора логов.

## Шаблонизация
Словари поддерживают динамические вставки:
*   `{{itemName}}` — подстановка названия предмета.
*   `{0}` — порядковая подстановка (например, количество предметов).

---

## AI-контекст
*   JSON файлы помечены как `linguist-generated`, чтобы не влиять на статистику кода.
*   При добавлении новой кнопки или текстового блока на фронтенде, необходимо добавить соответствующий ключ в оба файла, иначе пользователь увидит технический ID ключа.
