# [Frontend/Web/utils/SortController.ts](../../../Frontend/Web/utils/SortController.ts)

## Назначение
Универсальный контроллер для сортировки списков. Используется для упорядочивания предметов по редкости, стоимости или названию, а также героев по уровню.

## Связи (Dependencies)
*   `[global.d.ts](../types/global.md)`: Для понимания структуры данных, которые подлежат сортировке (Items, Heroes).

## Ключевая логика
Реализует паттерн «Стратегия». Хранит текущее направление (ASC/DESC) и ключ сортировки.

### Методы:
*   `sort(data, key, direction)`: Возвращает новый отсортированный массив.
*   **Специальная логика**: Содержит веса для редкостей (Common < Rare < Epic...), чтобы сортировка по редкости была логически верной для игрока.

## AI-контекст
*   **Иммутабельность**: Контроллер всегда возвращает копию массива. Это важно для реактивного обновления интерфейса.
*   **Расширение**: При добавлении новых типов данных (например, "Снаряжение") нужно добавить их веса в этот контроллер.
