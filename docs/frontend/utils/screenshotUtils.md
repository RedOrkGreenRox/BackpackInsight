# Утилиты скриншотов (`screenshotUtils.ts`)

## Назначение
Утилита для создания скриншотов конкретных областей интерфейса (например, рюкзака со сборкой). Это позволяет пользователям **Backpack Insight** делиться своими билдами в виде изображений.

## Связи (Dependencies)
*   `[global.d.ts](../types/global.md)`: Для типизации объектов DOM и параметров скриншота.
*   Использует внешнюю библиотеку `html2canvas` (или аналоги), которая должна быть упомянута в `package.json`.

## Ключевая логика
Функция захватывает целевой DOM-элемент, отрисовывает его на `canvas` и конвертирует в `Blob` или `DataURL`.

### Возможности:
*   `takeScreenshot(elementId, options)`: Основной метод. Позволяет скрыть ненужные элементы (кнопки управления) перед снимком.
*   **Оптимизация**: Автоматически подбирает масштаб для сохранения четкости иконок предметов из **Backpack Brawl**.

## AI-контекст
*   **Рендеринг**: При генерации новых компонентов (например, детальной карточки предмета), убедитесь, что они имеют четкие границы (ID или классы), чтобы эта утилита могла их корректно захватить.
*   **Проблемы**: Помните о CORS для изображений. Все иконки должны быть доступны для canvas, иначе скриншот будет пустым или с ошибкой.
