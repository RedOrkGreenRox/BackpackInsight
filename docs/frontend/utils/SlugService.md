# [Frontend/Web/ground/utils/SlugService.ts](../../../Frontend/Web/ground/utils/SlugService.ts)

## Назначение
Утилита для генерации «слагов» (безопасных URL-строк) из названий предметов и конвертации систем счисления.

## Связи (Dependencies)
*   Используется всеми сервисами, работающими с путями: `[ItemIconService.ts](./ItemIconService.md)`, `[ItemsCacheService.ts](./ItemsCacheService.md)`.

## Подробное описание методов

### `toSlug(name: string): string` (static)
Преобразует любую строку в формат, пригодный для URL и имен файлов.
*   **Пример**: "Wooden Sword +1" -> "wooden-sword-1".
*   Удаляет спецсимволы, заменяет пробелы на дефисы, нормализует Unicode.
*   Убирает повторяющиеся дефисы и дефисы в начале/конце.

### `romanToArabic(roman: string): number` (static)
Конвертирует римские цифры в арабские.
*   Поддерживает значения до 20 (от 'I' до 'XX').
*   Используется в **Backpack Brawl** для парсинга уровней чертежей и этапов ограблений.

### `isRomanNumeral(str: string): boolean` (static)
Проверяет, является ли строка известной римской цифрой.

## AI-контекст
*   **Единообразие**: Всегда используйте этот сервис для генерации ключей кеша или путей к изображениям. Не пишите свою логику замены пробелов на дефисы, иначе возникнет рассинхронизация с базой ассетов.
