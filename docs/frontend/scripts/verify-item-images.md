# [Проверка изображений предметов (verify-item-images.js)](../../../Frontend/Web/scripts/verify-item-images.js)

## Назначение
Инструмент (`npm run verify:images`): сверяет список предметов из `Backend/DB` с наличием файлов-иконок в `static/images/items`, чтобы выявить отсутствующие/лишние картинки.

## Связи (Dependencies)
*   Источник имён: JSON из `Backend/DB`.
*   Ассеты: `Frontend/Web/static/images/items`.
*   `MASKED_ITEMS` — те же исключения именования, что и в [ItemIconService](../ground/utils/ItemIconService.md) (предметы с чужой иконкой).

## Подробное описание
*   Строит ожидаемые слаги имён, проверяет наличие соответствующих файлов, репортит расхождения.
*   `MASKED_ITEMS` учитывает предметы, чьи иконки совпадают с другими (например, `Suspicious Sausage`→`tender-sausage`).

## AI-контекст
*   `MASKED_ITEMS` обязан совпадать с одноимённым словарём в `ItemIconService.ts` — при расхождении проверка даст ложные срабатывания.
