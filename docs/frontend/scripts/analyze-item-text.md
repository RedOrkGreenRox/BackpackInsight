# [Анализ текста предметов (analyze-item-text.js)](../../../Frontend/Web/scripts/analyze-item-text.js)

## Назначение
Инструмент-скрипт (Node, запускается `npm run analyze:item-text`): анализирует тексты описаний предметов из `Backend/DB`, строит частотный словарь токенов (с отсевом стоп-слов) — помогает настраивать семантический поиск.

## Связи (Dependencies)
*   Источник: JSON-файлы из `Backend/DB`.
*   Результат пишет в [tmp/item-text-statistics.json](../tmp/item-text-statistics.md).
*   Помогает наполнять словари для [SearchTermService](../ground/utils/SearchTermService.md).

## Подробное описание
*   Пути: `WEB_ROOT`, `PROJECT_ROOT`, `DB_DIR`, `OUT_DIR=tmp`.
*   `MIN_WORD_LENGTH=2`; `--top=N` (по умолчанию 250) — сколько топ-токенов вывести.
*   `MANUAL_STOPWORDS` — ручной список служебных/частых слов (en/ru), исключаемых из анализа.
*   Читает описания, токенизирует, считает частоты, выводит топ.

## AI-контекст
*   Это dev-утилита, не часть рантайма. Результат используется человеком для пополнения синонимов/стоп-слов семантического поиска.
