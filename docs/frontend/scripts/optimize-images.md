# [Оптимизация изображений (optimize-images.js)](../../../Frontend/Web/scripts/optimize-images.js)

## Назначение
Заготовка-скрипт оптимизации изображений перед сборкой (Node, CommonJS). Сейчас в основном логирует и перечисляет критические изображения; реальную оптимизацию (sharp/imagemin) предполагается добавить.

## Подробное описание
*   `optimizeImages()` — выводит сообщения, перечисляет `criticalImages` (logo/menu webp) для возможного preload.
*   Точка расширения: здесь предполагается интеграция генерации WebP/уменьшенных версий.

## AI-контекст
*   На текущий момент скрипт — заглушка/каркас. Реальные ассеты в проекте уже лежат в avif/webp; при добавлении сжатия именно сюда подключайте sharp/imagemin.
