# [Сервис иконок предметов (ItemIconService.ts)](../../../../Frontend/Web/ground/utils/ItemIconService.ts)

## Назначение
Централизованный сервис для сопоставления игровых предметов с их графическими ассетами. Он решает проблему нелинейного маппинга, когда название предмета не совпадает с именем файла изображения.

---

## Ключевая логика

### 1. Маскировка предметов (`MASKED_ITEMS`)
В игре Backpack Brawl некоторые предметы являются "скинами" других предметов или используют общие ассеты. Сервис содержит карту исключений (например, `Feral Cat` мапится на `black-cat`).

### 2. Логика Heist Plans (Планы ограбления)
Для предметов редкости `Special` реализован автоматический разбор тултипов. Если описание начинается со слова `Step`, сервис извлекает число (поддерживаются как арабские, так и римские цифры через `SlugService`) и возвращает путь к соответствующей иконке плана.

### 3. Дефолтный путь
Для стандартных предметов путь генерируется через [**SlugService**](SlugService.md) путем нормализации названия.

---

## AI-контекст
*   Этот сервис — **единый источник истины** для путей иконок. Его используют и каталог предметов, и страница профиля, и детальная карточка.
*   При добавлении новых "Special" предметов с уникальной логикой отображения, правки нужно вносить именно в `STEP_PATTERN`.
