# [Сервис форматов изображений (ImageFormatService.ts)](../../../../Frontend/Web/ground/utils/ImageFormatService.ts)

## Назначение
Критический системный сервис для оптимизации трафика. Он определяет возможности браузера пользователя и выбирает наиболее современный поддерживаемый формат изображений (AVIF или WebP).

---

## Ключевая логика

### 1. Проверка поддержки (`init`)
При старте приложения сервис проводит микро-тесты:
*   Пытается загрузить невидимый 1x1 пиксель в формате **AVIF**.
*   Если тест успешен — всё приложение переключается на AVIF.
*   Если нет — выполняется аналогичный тест для **WebP**.
*   Финальный выбор сохраняется в `document.documentElement.dataset['imageFormat']`.

### 2. Генерация путей (`itemSrc`)
Вместо того чтобы создавать тяжеловесные теги `<picture>` с множеством `<source>` для каждого из сотен предметов, приложение просто вызывает `itemSrc`. Сервис возвращает путь к одной картинке в идеальном для текущего браузера формате.

---

## AI-контекст
*   Сервис использует `Promise` с таймаутом 800мс для тестов, что предотвращает зависание инициализации на старых устройствах.
*   Использование единого формата для всех предметов значительно упрощает DOM и ускоряет рендеринг списков (Вики).
