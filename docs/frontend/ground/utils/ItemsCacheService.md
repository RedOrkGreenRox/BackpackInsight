# [Кэш предметов (ItemsCacheService.ts)](../../../../Frontend/Web/ground/utils/ItemsCacheService.ts)

## Назначение
Сервис для локального хранения данных о предметах. Он минимизирует количество запросов к API, сохраняя справочник во внутренней памяти приложения.

## Функционал
*   **Lazy Loading**: Запрашивает данные у `ApiService` только один раз при первом обращении.
*   **Словарь (Map)**: Предоставляет быстрый доступ к предмету по его ID.
