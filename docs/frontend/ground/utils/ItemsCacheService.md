# [Frontend/Web/ground/utils/ItemsCacheService.ts](../../../../Frontend/Web/ground/utils/ItemsCacheService.ts)

## Назначение
Единый кеш справочника предметов **Backpack Brawl**. Он оптимизирует работу приложения, объединяя запросы от разных страниц и сохраняя данные между перезагрузками в `sessionStorage`.

## Связи (Dependencies)
*   `[ApiService.ts](ApiService.md)`: Используется для фактической загрузки данных с сервера.
*   `[SlugService.ts](SlugService.md)`: Используется для поиска предметов в кеше по их слагу (URL-имени).

## Подробное описание методов и механизмов

### `getAllItems(): Promise<ItemDefinition[]>` (static)
Основной метод получения данных. Реализует многоуровневое хранение:
1.  **Memory Cache**: Если данные уже в переменной `memoryItems`, возвращает их мгновенно.
2.  **Storage Cache**: Если данных в памяти нет, пытается прочитать их из `sessionStorage`.
3.  **In-Flight Request**: Если данных нет нигде, делает запрос к API. 
    *   **Важно**: Если несколько компонентов вызовут этот метод одновременно, будет сделан только **один** сетевой запрос благодаря переменной `inFlight`. Все остальные вызовы «подпишутся» на этот же промис.

### `getBySlug(rawNameOrSlug)` (static)
Асинхронный поиск. Сначала гарантирует, что справочник загружен (через `getAllItems`), затем находит нужный предмет.

### `clear()` (static)
Полностью очищает память и хранилище. Используется при смене версии игры или принудительном обновлении данных.

### `read/writeToSessionStorage` (private)
Методы для сериализации массива предметов в JSON. Обрабатывают ошибки `QuotaExceededError` (если место в хранилище закончилось), чтобы приложение не упало.

## AI-контекст
*   **Синхронизация**: Этот сервис гарантирует, что на странице поиска и на странице деталей предмета отображаются одни и те же данные одной и той же версии.
*   **Производительность**: Если данные уже загружены, используйте синхронные `getAllItemsFromCache()` / `getBySlugFromCache(slug)` (возвращают `null`/`undefined` при отсутствии), не дожидаясь сетевого `getAllItems()`.
