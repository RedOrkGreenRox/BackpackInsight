# [SEO и Мета-данные (MetaService.ts)](../../../../Frontend/Web/ground/utils/MetaService.ts)

## Назначение
Сервис для динамического управления тегами внутри секции `<head>`. Обеспечивает корректное SEO и отображение сайта в социальных сетях без полной перезагрузки страницы (в рамках SPA).

---

## Возможности

### 1. Мета-теги
Метод `setMeta` позволяет налету менять описание (`description`), ключевые слова (`keywords`) и свойства OpenGraph (например, `og:image`). Если тег отсутствует, сервис создает его автоматически.

### 2. Управление ссылками
Метод `setLink` управляет каноническими адресами (`canonical`) и языковыми альтернативами (`hreflang`), что критично для предотвращения дублирования контента в поисковиках.

### 3. Структурированные данные (JSON-LD)
Сервис умеет динамически вставлять скрипты `application/ld+json`. Это используется для предоставления поисковым роботам детальной информации о предметах (стоимость, редкость, бренд) в машиночитаемом виде.

---

## AI-контекст
*   Сервис гарантирует, что каждая страница SPA будет иметь уникальный `document.title`, который видит пользователь в заголовке вкладки.
