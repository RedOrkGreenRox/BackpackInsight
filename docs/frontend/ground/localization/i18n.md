# [Локализация (i18n.ts)](../../../../Frontend/Web/ground/localization/i18n.ts)

## Назначение
Менеджер многоязычности. Отвечает за загрузку словарей и перевод строк интерфейса.

## Возможности
*   **Автоопределение**: Выбирает язык на основе настроек браузера пользователя.
*   **Динамическая замена**: Поддерживает вставку переменных в текст (например, `{{count}}`).
*   **Переключение**: Позволяет менять язык "на лету" без перезагрузки всей страницы.
