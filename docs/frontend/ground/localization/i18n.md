# [Frontend/Web/ground/localization/i18n.ts](../../../../Frontend/Web/ground/localization/i18n.ts)

## Назначение
Настройка интернационализации (i18next) для **Backpack Brawl**. Позволяет переключать язык интерфейса (RU/EN) «на лету» без перезагрузки приложения.

## Связи (Dependencies)
*   **JSON Словари**: Загружает данные из `Frontend/Web/lang/ru.json` и `en.json`.
*   `[sidebar.ts](../roots/_roots/shell/sidebar/sidebar.md)`: Компонент сайдбара содержит переключатель языков, который вызывает методы этого файла.

## Ключевая логика
Файл инициализирует библиотеку `i18next`, настраивает детектор языка пользователя (LanguageDetector) и способ загрузки ресурсов.

### Основные функции:
*   `changeLanguage(lang)`: Сменяет текущую локаль и уведомляет приложение об изменениях.
*   `t(key)`: Функция-переводчик, возвращающая строку по ключу из JSON-словаря.

## AI-контекст
*   **Локализация контента**: Все текстовые строки в компонентах (бранчах) должны выводиться через функцию `t`. Хардкод строк запрещен для соблюдения «гигиены» проекта.
*   **Предметы**: Обратите внимание, что названия самих предметов часто приходят с бэкенда уже локализованными или требуют особого ключа для перевода в `items_tooltips.json`.
