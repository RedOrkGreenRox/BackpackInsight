# [Frontend/Web/ground/roots/Branch.ts](../../../../Frontend/Web/ground/roots/Branch.ts)

## Назначение
Базовый абстрактный класс для всех страниц (компонентов верхнего уровня) приложения. Он задает стандартный жизненный цикл для любого раздела — от главной страницы до настроек.

## Связи (Dependencies)
*   `[Shell.ts](./Shell.md)`: Любой `Branch` инициализируется внутри «оболочки».
*   `[eventUtils.ts](../../utils/eventUtils.md)`: Базовый класс предоставляет методы для удобной подписки на системные события.

## Жизненный цикл (Lifecycle)
*   `init()`: Начальная загрузка данных и настройка слушателей.
*   `render()`: Отрисовка HTML-структуры бранча.
*   `destroy()`: Очистка памяти, удаление обработчиков и DOM-элементов.

## AI-контекст
*   **Наследование**: Все новые разделы проекта **Backpack Insight** должны наследоваться от этого класса. Никогда не создавайте страницы «с нуля».
*   **Архитектура**: Это ядро концепции «Бранчей». Если нужно изменить поведение всех страниц разом (например, добавить логгер), это делается здесь.
