# [Глобальные анимации (_animations.scss)](../../../../Frontend/Web/ground/roots/_roots/_animations.scss)

## Назначение
Центральное хранилище всех CSS-анимаций (keyframes) проекта. Здесь описаны эффекты появления карточек, пульсация кнопок и свечение редких предметов.

## Связи (Dependencies)
*   Используется всеми бранчами, например [Стилями профиля](../../branches/profile/profile.md).

## Ключевая логика
*   `@keyframes fadeIn`: Базовая анимация появления элементов.
*   `@keyframes pulse`: Эффект для привлечения внимания к важным кнопкам.
*   `@keyframes shimmmer`: Скелетон-загрузка (мерцание пустых блоков).

## AI-контекст
Для управления анимациями из JS используется библиотека AOS, но базовые низкоуровневые эффекты определены именно здесь.
