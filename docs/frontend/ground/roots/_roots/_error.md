# [Стили ошибок (_error.scss)](../../../../../Frontend/Web/ground/roots/_roots/_error.scss)

## Назначение
Описание визуального оформления системных уведомлений об ошибках.

## Основной класс: `.error`
*   **Цвет**: Насыщенный красный фон (`#c0392b`).
*   **Оформление**: Скругленные углы (15px) и внутренняя обводка.
*   **Использование**: Применяется для вывода сообщений о недоступности бэкенда или критических сбоях парсинга.
