# [Вкладки навигации (_nav-tab.scss)](../../../../../../../Frontend/Web/ground/roots/_roots/shell/sidebar/_nav-tab.scss)

## Назначение
Стилизация элементов списка меню (Главная, Предметы и др.).

---

## Ключевые элементы

### `.nav-tab`
*   **Макет**: Горизонтальный flex с адаптивными отступами.
*   **Эффект выделения**: Слева добавляется цветная полоса (`border-left`), если вкладка активна.
*   **Градиент**: По наведению или при активности за вкладкой появляется мягкое радиальное свечение в акцентном цвете.

### Текст и Иконки
*   **Иконки**: Используют `clamp(32px, 5vmin, 40px)` для идеального масштабирования между мобильными и десктопами.
*   **Активное состояние**: Текст получает тень `text-shadow` для эффекта свечения.
