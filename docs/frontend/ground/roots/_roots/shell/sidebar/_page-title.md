# [Заголовки меню (_page-title.scss)](../../../../../../../Frontend/Web/ground/roots/_roots/shell/sidebar/_page-title.scss)

## Назначение
Стилизация текстовых меток внутри навигационных вкладок.

## Характеристики
*   **Типографика**: Жирное начертание (700), размер 1.5em.
*   **Чтение**: Добавлена тень `0 2px 8px rgba(0, 0, 0, 0.6)` для обеспечения читаемости поверх динамических фонов.
*   **Мобильность**: На экранах уже 600px размер шрифта автоматически уменьшается до `small`.
