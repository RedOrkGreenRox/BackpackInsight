# [Боковое меню (_sidebar.scss)](../../../../../../../Frontend/Web/ground/roots/_roots/shell/sidebar/_sidebar.scss)

## Назначение
Описание структуры и анимации основной панели навигации (Sidebar).

---

## Ключевая логика

### 1. Позиционирование и Размеры
*   Панель всегда прижата к правому краю (`right: 0`).
*   **Ширина**: Адаптивная, от 240px до 350px на десктопе, и до 85% ширины экрана на мобильных устройствах.

### 2. Анимация открытия
Используется `transform: translateX(100%)` для скрытия панели за правой границей экрана. При добавлении класса `.open`, панель плавно выезжает, используя сложную кривую `cubic-bezier(0.19, 1, 0.22, 1)`, что создает эффект инерции.

### 3. Оверлей (`.sidebar-overlay`)
При открытии меню весь остальной контент затемняется полупрозрачным фоном (`rgba(0, 0, 0, 0.6)`), который блокирует взаимодействие с основной страницей.
