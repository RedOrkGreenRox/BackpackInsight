# [Логотип в меню (_button-logo.scss)](../../../../../../../Frontend/Web/ground/roots/_roots/shell/sidebar/_button-logo.scss)

## Назначение
Стилизация кнопки-логотипа, расположенной в верхней части бокового меню.

## Визуальные характеристики
*   **Размер**: 128x128px.
*   **Интерактивность**: При наведении логотип плавно увеличивается (`scale(1.05)`).
*   **Контейнер**: Использует `<picture>` для поддержки современных графических форматов.
