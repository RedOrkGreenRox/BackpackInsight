# [Инициализация UI (ui_init.ts)](../../../../../../../Frontend/Web/ground/roots/_roots/shell/ui_init/ui_init.ts)

## Назначение
Скрипт первичной подготовки DOM-дерева. Он отвечает за мгновенное отображение текстовых блоков и настройку ленивой загрузки изображений.

---

## Функционал

### 1. Обработка ошибок изображений
Глобальная функция `handleImageError` перехватывает сбои загрузки иконок:
*   Удаляет теги `<source>` из родительского `<picture>`.
*   Устанавливает стандартный плейсхолдер предмета.

### 2. Оптимизация рендеринга
*   **Мгновенный показ**: Применяет класс `.loaded` к текстовым элементам и кнопкам сразу после `DOMContentLoaded`, предотвращая эффект "пустого экрана".
*   **Lazy Loading**: Автоматически проставляет атрибут `loading="lazy"` всем изображениям, кроме критических (логотип и фон).

### 3. Состояние меню
Подсвечивает активный пункт в сайдбаре, сравнивая текущий `location.pathname` с атрибутами вкладок.

---

## Связи (Dependencies)
*   Использует [**ImageFormatService**](../../../../utils/ImageFormatService.md) для получения пути к плейсхолдеру.
