# [Frontend/Web/ground/roots/_roots/shell/ui_init/ui_init.ts](../../../../../../Frontend/Web/ground/roots/_roots/shell/ui_init/ui_init.ts)

## Назначение
Этот файл выполняет черновую работу по первичной инициализации UI: установку глобальных слушателей (resize, scroll), детектирование типа устройства и настройку CSS-переменных высоты экрана.

## Связи (Dependencies)
*   `[Shell.ts](../../../../Shell.md)`: Вызывается в самом начале процесса сборки оболочки.
*   `[_vars.scss](../../../../_roots/_vars.md)`: Управляет динамическими значениями, которые могут переопределять переменные из этого файла.

## Ключевая логика
Решает проблемы кроссбраузерности (например, корректный расчет `100vh` на мобильных устройствах с учетом адресной строки).
