# [Кнопка меню (_button-toggle.scss)](../../../../../../../Frontend/Web/ground/roots/_roots/shell/navigation/_button-toggle.scss)

## Назначение
Стилизация основной кнопки открытия меню (бургера). Файл определяет внешний вид, анимацию и поведение кнопки в зависимости от состояния приложения.

---

## Визуальные характеристики
*   **Размер**: Фиксированный 100x100px.
*   **Форма**: Квадрат со скруглением 20px и стандартной рамкой `--border`.
*   **Фон**: Полупрозрачный черный (`rgba(0, 0, 0, 0.4)`).

## Интерактивность
*   **Ховер**: Иконка внутри кнопки плавно увеличивается (`scale(1.1)`).
*   **Состояние открытого меню**: Когда `body.sidebar-open` активно, кнопка становится неклиткабельной (`pointer-events: none`) и плавно "уезжает" вправо за пределы экрана, синхронизируясь с анимацией сайдбара.

---

## Связи (Dependencies)
*   Используется в [**navigation.ts**](../../navigation.md) для управления UI.
*   Подключается в [**_shell.scss**](../../_shell.md).
