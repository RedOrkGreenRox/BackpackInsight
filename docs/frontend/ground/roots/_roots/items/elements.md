# [Стили элементов предмета (_item-name.scss / _item-level.scss / _item-rarities.scss)](../../../../../../Frontend/Web/ground/roots/_roots/items/_item-name.scss)

## Назначение
Детальные стили для внутренних элементов карточки предмета.

## Описание
*   **`_item-name.scss`**: Стилизация заголовка с поддержкой многострочного текста.
*   **`_item-level.scss`**: Дизайн значка уровня предмета (круглый индикатор в углу).
*   **`_item-rarities.scss`**: Глобальная карта цветов для всех уровней редкости (от обычного серого до золотого `Godly`).
