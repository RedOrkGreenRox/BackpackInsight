# [Переменные редкости (_rarity-vars.scss)](../../../../../../Frontend/Web/ground/roots/_roots/items/_rarity-vars.scss)

## Назначение
Этот файл определяет цветовую палитру редкостей предметов, используемую для окрашивания текста, рамок и эффектов свечения. Он является единственным источником истины для визуального разделения уровней силы предметов.

---

## Список переменных

### Базовые цвета (`--rarity-*-base`)
*   **Common**: `#EFEFEF` (белый/серый).
*   **Rare**: `#AFE5FF` (голубой).
*   **Epic**: `#E59FFF` (фиолетовый).
*   **Legendary**: `#FF8E56` (оранжевый).
*   **Mythic**: `#ff5555` (красный).
*   **Unique**: `#00ffaa` (бирюзовый).
*   **Relic**: `#ff69b4` (розовый).
*   **Boon**: `#FFD19A` (светло-оранжевый).
*   **Special**: `#FFEB7A` (желтый).

### Производные переменные
Для каждой редкости генерируются три типа переменных через `color-mix`:
*   `--rarity-*-text`: Цвет для текста.
*   `--rarity-*-glow`: Цвет для теней и свечений.
*   `--rarity-*-bg`: Цвет фона (смешивание с черным 60%).

---

## Связи (Dependencies)
*   Используется в [**_item-rarities.scss**](elements.md) для применения стилей к конкретным классам.
*   Подключается в [**_items.scss**](base.md).

## AI-контекст
*   Переменная `--rarity-border-unified` (`#321F19`) задает одинаковый коричневый оттенок для рамок всех предметов, что объединяет их в едином стиле игры.
