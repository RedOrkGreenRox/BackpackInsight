# [Сетка предметов (_items-grid.scss)](../../../../../../Frontend/Web/ground/roots/_roots/items/_items-grid.scss)

## Назначение
Описание макета (Layout) для отображения списков предметов. Использует CSS Grid для создания адаптивной и производительной сетки.

---

## Конфигурация сетки

### Основные параметры
*   **Дисплей**: `grid`.
*   **Колонки**: Ровно 5 колонок (`repeat(5, 1fr)`) на десктопах.
*   **Ограничение ширины**: Максимум `1000px`, чтобы сетка не растягивалась на сверхшироких мониторах.

### Адаптивность (Responsive)
Сетка динамически меняет количество колонок в зависимости от ширины экрана:
*   **< 900px**: 4 колонки.
*   **< 700px**: 3 колонки.
*   **< 480px**: 2 колонки (мобильный режим) с уменьшенным зазором (10px).

---

## Связи (Dependencies)
*   Подключается в [**_items.scss**](base.md).
*   Применяется к контейнерам предметов в Библиотеке и Профиле игрока.
