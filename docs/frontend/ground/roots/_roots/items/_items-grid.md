# [Стили сетки рюкзака (_items-grid.scss)](../../../../../../Frontend/Web/ground/roots/_roots/items/_items-grid.scss)

## Назначение
Адаптивная CSS Grid-сетка `.items-grid` для отображения карточек предметов (рюкзак, список). Глобальный «корневой» компонент, переиспользуемый разными страницами.

## Задаваемые стили

### `.items-grid` (десктоп)
*   `display: grid`; `grid-template-columns: repeat(5, minmax(0, 1fr))` — ровно 5 колонок.
*   `gap: 15px`.
*   Центрирование/ширина: `width: min(100%, 1000px); max-width:1000px; margin-inline:auto; box-sizing:border-box`.

### Адаптивность (`@media`)
*   `≤900px`: 4 колонки (`repeat(4, minmax(0,1fr))`).
*   `≤700px`: 3 колонки.
*   `≤480px`: 2 колонки + уменьшенный `gap:10px`.

## AI-контекст
*   `minmax(0, 1fr)` намеренно вместо `1fr` — предотвращает переполнение сетки длинным контентом. Количество колонок задаётся пошагово брейкпоинтами, без `auto-fill`.
