# [Стиль карточки предмета (_item-card.scss / _item-image.scss)](../../../../../../Frontend/Web/ground/roots/_roots/items/_item-card.scss)

## Назначение
Базовые стили для отображения предметов во всех частях сайта (библиотека, профиль, детали).

## Ключевые компоненты
*   **`.item-card`**: Обертка с эффектом глубины и наведения.
*   **`.item-image`**: Контейнер для иконки с поддержкой адаптивного размера и ленивой загрузки.
*   **`.item-rarity-*`**: Динамические цвета рамок и свечений, зависящие от редкости предмета (Common, Rare, Epic и т.д.).
