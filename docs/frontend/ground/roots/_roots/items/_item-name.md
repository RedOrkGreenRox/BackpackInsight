# [Название предмета (_item-name.scss)](../../../../../../Frontend/Web/ground/roots/_roots/items/_item-name.scss)

## Назначение
Стилизация текстового заголовка предмета. Файл обеспечивает читаемость имен разной длины.

---

## Ключевые правила
*   **Типографика**: Размер 0.9rem, средний вес (500), цвет светло-серый.
*   **Ограничение длины**: Используется `line-clamp: 2`. Если название предмета слишком длинное, оно автоматически обрезается троеточием после второй строки.
*   **Выравнивание**: Текст всегда центрирован.

---

## Связи (Dependencies)
*   Подключается в [**_items.scss**](base.md).
