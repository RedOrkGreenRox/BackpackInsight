# [Уровень предмета (_item-level.scss)](../../../../../../Frontend/Web/ground/roots/_roots/items/_item-level.scss)

## Назначение
Минималистичные стили для отображения уровня предмета (LVL) на карточке.

## Характеристики
*   **Шрифт**: Мелкий (small), тонкий (300).
*   **Цвет**: Использует переменную `--text-default-color`, обеспечивая мягкий контраст.
