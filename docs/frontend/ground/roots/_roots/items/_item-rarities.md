# [Стили редкостей (_item-rarities.scss)](../../../../../../Frontend/Web/ground/roots/_roots/items/_item-rarities.scss)

## Назначение
Связующий файл, который назначает цвета из переменных конкретным CSS-классам редкостей.

---

## Реализация
Для каждого типа редкости (`common`, `rare`, `epic`, `legendary`, `mythic`, `unique`, `relic`, `boon`, `special`) файл задает:
1.  **Цвет текста** (`color`): берется из соответствующих переменных `--rarity-*-text`.
2.  **Цвет свечения** (`--rarity-glow-color`): локальная переменная, которая пробрасывается дочерним элементам для использования в тенях и обводках.

---

## Связи (Dependencies)
*   Зависит от [**_rarity-vars.scss**](_rarity-vars.md).
*   Подключается в [**_items.scss**](base.md).
