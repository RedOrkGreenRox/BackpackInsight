# [Дизайн-система: Переменные (_vars.scss)](../../../../../Frontend/Web/ground/roots/_roots/_vars.scss)

## Назначение
Главный файл дизайна. Определяет цвета редкостей, сетку и адаптивность.

## Связи (Dependencies)
Используется **всеми** `.scss` файлами проекта, например [Стилями профиля](../../branches/profile/profile.md).

## Ключевые переменные
*   `$legendary`, `$mythic`: Цвета для редкостей предметов **Backpack Brawl**.
*   `$breakpoint-mobile`: Граница для адаптивной верстки.

## AI-контекст
Никогда не используйте "сырые" цвета (hex/rgb). Всегда берите переменные из этого файла для соблюдения гигиены кода.
