# [Сброс стилей (_reset.scss)](../../../../../Frontend/Web/ground/roots/_roots/_reset.scss)

## Назначение
Базовый файл для нормализации стилей во всех браузерах. Он удаляет стандартные отступы, настраивает `box-sizing` и подготавливает «чистый холст» для дизайна **Backpack Brawl**.

## Связи (Dependencies)
*   Подключается в [Корневой файл стилей](../../../../../Frontend/Web/ground/roots/_roots.scss).

## Ключевая логика
*   **Box-sizing**: Устанавливает `border-box` для всех элементов, что критично для правильного расчета размеров слотов рюкзака.
*   **Typography**: Убирает стандартные стили списков и ссылок.
*   **Scroll**: Настраивает плавную прокрутку (`scroll-behavior: smooth`).
