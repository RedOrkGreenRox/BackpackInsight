# [Глобальные шрифты (_fonts.scss)](../../../../../Frontend/Web/ground/roots/_roots/_fonts.scss)

## Назначение
Этот файл отвечает за подключение и настройку типографики проекта. Он определяет шрифты, которые создают атмосферу RPG в **Backpack Insight**.

## Связи (Dependencies)
*   **Ассеты**: Ссылается на файлы шрифтов в `/static/fonts/`.
*   [Переменные дизайна](_vars.md): Использует переменные для задания базовых размеров текста.

## Ключевая логика
*   **Font-face**: Регистрирует локальные шрифты (например, Noto Sans, Signika), обеспечивая работу сайта без интернета (PWA).
*   **Антиалиасинг**: Настраивает `font-smoothing` для четкого отображения текста на Retina-дисплеях.

## AI-контекст
При добавлении новых стилей текста всегда используйте шрифтовые семейства, объявленные здесь, чтобы сохранять визуальное единство с игрой.
