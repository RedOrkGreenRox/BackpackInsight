# [Подключение шрифтов (_fonts.scss)](../../../../../Frontend/Web/ground/roots/_roots/_fonts.scss)

## Назначение
Этот файл отвечает за загрузку и оптимизацию шрифтов, используемых в проекте. Он гарантирует правильное отображение текста на разных языках и начертаниях.

---

## Используемые шрифты

### 1. Signika
Основной шрифт для латинских символов. Используется для имен предметов, цифр и игровых терминов.
*   **Веса**: 400 (Regular), 700 (Bold).
*   **Диапазон**: `U+0000-00FF` (базовая латиница).
*   **Формат**: `woff2`.

### 2. Noto Sans
Шрифт для кириллических символов (русский язык).
*   **Веса**: 400, 700.
*   **Диапазон**: `U+0400-04FF` (кириллица).
*   **Зачем**: `Signika` не поддерживает русский язык, поэтому для корректного отображения локализации используется качественный фоллбэк.

---

## Оптимизация
*   **`font-display: swap`**: Позволяет браузеру сразу показывать текст системным шрифтом, пока грузятся основные файлы, предотвращая невидимый текст (FOIT).
*   **`unicode-range`**: Позволяет не грузить кириллический шрифт пользователям, которые используют только английский интерфейс.

## Связи (Dependencies)
*   Подгружается через [`_roots.scss`](../_roots.md).
*   Файлы шрифтов находятся в `/static/fonts/`.
