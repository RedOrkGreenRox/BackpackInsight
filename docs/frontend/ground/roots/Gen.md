# [Frontend/Web/ground/roots/Gen.ts](../../../../Frontend/Web/ground/roots/Gen.ts)

## Назначение
«Генератор» или Фабрика компонентов. Этот файл содержит статические методы для быстрого создания типичных элементов интерфейса (кнопки, карточки предметов, заголовки) в едином стиле **Backpack Brawl**.

## Связи (Dependencies)
*   `[_vars.scss](./_roots/_vars.md)`: Использует глобальные CSS-классы и переменные для стилизации генерируемых элементов.
*   `[i18n.ts](../localization/i18n.md)`: Автоматически локализует создаваемые элементы интерфейса.

## Ключевая логика
Вместо написания HTML-строк вручную, проект использует методы `Gen` для сборки DOM-дерева. Это обеспечивает «гигиену» кода и предотвращает XSS-атаки.

### Примеры методов:
*   `btn(text, callback, type)`: Создает стилизованную кнопку.
*   `itemCard(itemData)`: Генерирует сложную карточку предмета со всеми иконками.

## AI-контекст
*   **Стандартизация**: При создании новых UI-элементов ИИ должен сначала проверить наличие подходящего метода в `Gen.ts`. Добавление нового метода в `Gen` предпочтительнее, чем дублирование верстки в разных бранчах.
