# [Макет деталей предмета (_layout.scss)](../../../../../../../Frontend/Web/ground/branches/itemDetail/_itemDetail/layout/_layout.scss)

## Назначение
Описание структуры контейнера страницы предмета.

## Характеристики
*   **Контейнер**: Имеет фиксированный `min-height: 100vh` для обеспечения правильной работы параллакса.
*   **Карточка**: Центрированный блок с полупрозрачным фоном (`0.7`) и неоновым градиентом сверху.
*   **Анимация**: Плавное появление всей карточки при загрузке.
