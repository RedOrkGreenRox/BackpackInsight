# [Блок информации о предмете (_info.scss)](../../../../../../../Frontend/Web/ground/branches/itemDetail/_itemDetail/info/_info.scss)

## Назначение
Стили для текстового блока описания предмета (flavor text и основные статы).

## Ключевая логика
Задает типографику для названия предмета и его редкости. Использует `opacity` для менее важных текстовых приписок.
