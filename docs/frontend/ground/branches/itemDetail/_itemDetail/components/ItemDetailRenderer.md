# [Рендерер деталей предмета (ItemDetailRenderer.ts)](../../../../../../../Frontend/Web/ground/branches/itemDetail/_itemDetail/components/ItemDetailRenderer.ts)

## Назначение
Статический класс для генерации HTML-разметки страницы детального просмотра предмета. Содержит логику сборки сложных визуальных элементов на основе игровых данных.

---

## Ключевая логика

### 1. Полностраничный рендеринг (`renderFullPage`)
Собирает интерфейс из функциональных блоков:
*   **Навигация**: Кнопки «Вперед/Назад» и «К списку», сформированные на основе текущего контекста (Вики или Профиль).
*   **Заголовок**: Комбинация названия, стоимости в золоте и иконки связанного героя.
*   **Визуализация**: Портрет предмета в рамке соответствующей редкости.
*   **Прогресс игрока**: (опционально) Отрисовывает текущий уровень и количество карт, если пользователь перешел с экрана аналитики профиля.

### 2. Динамическое SEO (`getMeta`)
Метод извлекает наиболее чистое название предмета из доступных данных (itemData, playerItem или URL) для формирования заголовка вкладки браузера.

### 3. Обработка характеристик (`renderWikiInfo`)
*   **Combat Stats**: Итерирует по объекту боевых параметров и заменяет ключи (напр. `damageMin`) на соответствующие иконки через [**icon-parser**](../../../../utils/icon-parser.md).
*   **Описание**: Объединяет массив тултипов в единый текст с автоматической подстановкой иконок механик (Burn, Poison и т.д.).
