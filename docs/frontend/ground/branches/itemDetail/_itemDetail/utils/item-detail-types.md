# [Типы данных деталей (item-detail-types.ts)](../../../../../../../Frontend/Web/ground/branches/itemDetail/_itemDetail/utils/item-detail-types.ts)

## Назначение
Описание TypeScript-интерфейсов для страницы глубокого анализа предметов.

---

## Ключевые интерфейсы
*   **`ItemDetailData`**: Полный объект состояния страницы (данные Wiki + динамические данные игрока).
*   **`NavigationState`**: Описание ссылок на предыдущий и следующий предметы.
*   **`PlayerItemData`**: Структура данных о прогрессе прокачки (LVL, Cards).
