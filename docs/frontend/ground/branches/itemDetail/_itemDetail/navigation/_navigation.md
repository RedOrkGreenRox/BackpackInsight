# [Навигация в деталях (_navigation.scss)](../../../../../../../Frontend/Web/ground/branches/itemDetail/_itemDetail/navigation/_navigation.scss)

## Назначение
Стили для кнопок "Предыдущий предмет" и "Следующий предмет".

## Связи (Dependencies)
*   [Менеджер навигации по предметам](../managers/ItemNavigationManager.md): Управляет появлением этих кнопок.

## Ключевая логика
Позиционирует кнопки по краям экрана или по бокам от карточки предмета. Содержит стили для иконок-стрелок.
