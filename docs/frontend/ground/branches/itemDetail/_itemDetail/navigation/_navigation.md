# [Верхняя навигация (_navigation.scss)](../../../../../../../Frontend/Web/ground/branches/itemDetail/_itemDetail/navigation/_navigation.scss)

## Назначение
Стилизация кнопок навигации «Вперед/Назад/К списку» в деталях предмета.

## Характеристики
*   **Sticky**: Панель фиксируется при скролле (`position: sticky`), позволяя быстро переключаться между предметами при чтении длинных описаний.
*   **Дизайн**: Кнопки в стиле Glass UI с темным фоном (`0.9`) и акцентной рамкой.
*   **Состояния**: Кнопки плавно приподнимаются при наведении. Инвалидные кнопки (`.disabled`) становятся полупрозрачными.
