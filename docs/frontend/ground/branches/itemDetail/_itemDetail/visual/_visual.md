# [Визуализация предмета (_visual.scss)](../../../../../../../Frontend/Web/ground/branches/itemDetail/_itemDetail/visual/_visual.scss)

## Назначение
Стилизация центрального блока с изображением предмета. Создает эффект "свечения" редкости за иконкой.

## Характеристики
*   **Свечение**: Использует `::after` с радиальным градиентом. Цвет берется из переменной `--rarity-glow-color`.
*   **Изображение**: Размер 160x160px, имеет глубокую тень `drop-shadow` для создания 3D-эффекта.
*   **Интерактивность**: При наведении (через родительский hover) свечение плавно усиливается.
