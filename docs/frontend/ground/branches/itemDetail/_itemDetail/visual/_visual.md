# [Визуальное представление предмета (_visual.scss)](../../../../../../../Frontend/Web/ground/branches/itemDetail/_itemDetail/visual/_visual.scss)

## Назначение
Стили для большого изображения предмета в детальном просмотре.

## Ключевая логика
Обеспечивает корректное масштабирование иконки предмета. Добавляет декоративные эффекты свечения (box-shadow) в зависимости от редкости.
