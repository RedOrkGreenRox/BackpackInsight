# [Контейнер поиска (_container.scss)](../../../../../../../Frontend/Web/ground/branches/items/_items/search/_container.scss)

## Назначение
Стили обёртки строки поиска `.search-container` на странице списка предметов.

## Задаваемые стили

### `.search-container`
*   `margin: 20px auto 40px` — центрирование по горизонтали + вертикальные отступы (сверху 20px, снизу 40px).
*   `max-width: 600px` — ограничение ширины поля поиска.
*   `position: relative` — система координат для возможного выпадающего списка подсказок.
*   `z-index: 10` — поле поиска над соседним контентом.

## AI-контекст
*   `position:relative; z-index:10` зарезервированы под абсолютно позиционируемые подсказки/автокомплит внутри контейнера.
