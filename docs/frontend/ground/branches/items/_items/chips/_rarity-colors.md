# [Цвета редкостей чипсов (_rarity-colors.scss)](../../../../../../../Frontend/Web/ground/branches/items/_items/chips/_rarity-colors.scss)

## Назначение
Массив классов для окраски элементов фильтров (чипсов) в цвета соответствующих редкостей предметов.

## Связи (Dependencies)
*   [Переменные дизайна](../../../../roots/_roots/_vars.md): Берет эталонные цвета оттуда.

## Ключевая логика
Определяет классы `.rarity-common`, `.rarity-legendary` и т.д. Эти классы используются для динамической окраски кнопок фильтров.
