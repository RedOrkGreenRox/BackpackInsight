# [Фильтр-чипсы (_filter-chip.scss)](../../../../../../../Frontend/Web/ground/branches/items/_items/chips/_filter-chip.scss)

## Назначение
Описание универсального интерактивного элемента ("чипсы") для выбора параметров фильтрации.

---

## Особенности реализации

### 1. Состояния
*   **Базовое**: Полупрозрачная серая плашка.
*   **Hover**: Легкий подъем и осветление фона.
*   **Active**: Полная заливка акцентным цветом `--azure` с эффектом свечения.

### 2. Поддержка иконок
Внутри чипсов могут находиться:
*   **Текст**: Для простых параметров (тип, редкость).
*   **Иконки**: Для героев и баффов. Размер иконок автоматически увеличивается до 32px в блоке выбора персонажей.
*   **Комбинация**: Иконка + Текст.

### 3. Форматирование
Использует `display: inline-flex` и `white-space: nowrap`, что гарантирует целостность чипса при переносе на новую строку.
