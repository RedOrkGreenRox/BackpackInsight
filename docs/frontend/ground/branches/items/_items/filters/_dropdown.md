# [Контейнер выпадающего списка (_dropdown.scss)](../../../../../../../Frontend/Web/ground/branches/items/_items/filters/_dropdown.scss)

## Назначение
Стилизация обертки для отдельного фильтра-дропдауна.

## Характеристики
*   **Макет**: Использует `flex: 1` для равномерного распределения нескольких фильтров в одном ряду.
*   **Безопасность**: `min-width: 0` предотвращает выход контента за границы flex-контейнера при переполнении.
