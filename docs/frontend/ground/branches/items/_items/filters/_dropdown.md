# [Контейнер dropdown-фильтра (_dropdown.scss)](../../../../../../../Frontend/Web/ground/branches/items/_items/filters/_dropdown.scss)

## Назначение
Обёртка `.dropdown-filter` для одного выпадающего фильтра в ряду.

## Задаваемые стили
### `.dropdown-filter`
*   `position:relative` — координаты для абсолютного контента dropdown.
*   `margin-bottom:20px`.
*   `flex:1; min-width:0` — равномерно делит ширину ряда, не переполняясь.

## AI-контекст
*   `min-width:0` критичен во flex-ряду, иначе длинный контент распирал бы колонку.
