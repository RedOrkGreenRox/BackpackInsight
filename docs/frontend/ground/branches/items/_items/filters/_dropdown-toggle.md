# [Кнопка-переключатель списка (_dropdown-toggle.scss)](../../../../../../../Frontend/Web/ground/branches/items/_items/filters/_dropdown-toggle.scss)

## Назначение
Стилизация кнопки, которая открывает и закрывает конкретный фильтр.

---

## Интерактивность
*   **Состояние Open**: При открытии кнопка получает подсветку `--azure`, а иконка стрелки (`.dropdown-arrow`) поворачивается на 180 градусов.
*   **Дизайн**: Использует верхнее скругление (8px 8px 0 0), чтобы бесшовно соединяться с выпадающим списком.
