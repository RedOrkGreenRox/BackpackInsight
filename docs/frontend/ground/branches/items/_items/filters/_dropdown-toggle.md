# [Переключатель выпадающего списка (_dropdown-toggle.scss)](../../../../../../../Frontend/Web/ground/branches/items/_items/filters/_dropdown-toggle.scss)

## Назначение
Стиль кнопки, по которой пользователь кликает для открытия фильтра.

## Ключевая логика
Содержит стили для иконки-стрелочки, которая поворачивается на 180 градусов при открытии списка.
