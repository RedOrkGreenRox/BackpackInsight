# [Контент выпадающего фильтра (_dropdown-content.scss)](../../../../../../../Frontend/Web/ground/branches/items/_items/filters/_dropdown-content.scss)

## Назначение
Стили для внутренней части выпадающего списка фильтров, где расположены кнопки-чипсы.

## Ключевая логика
Задает максимальную высоту блока и включает внутреннюю прокрутку (`overflow-y: auto`), если опций фильтра слишком много.
