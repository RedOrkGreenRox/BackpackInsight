# [Контент выпадающего списка (_dropdown-content.scss)](../../../../../../../Frontend/Web/ground/branches/items/_items/filters/_dropdown-content.scss)

## Назначение
Стилизация области с вариантами выбора внутри фильтра.

---

## Особенности реализации
*   **Скролл**: Индивидуальный брендированный скроллбар с ползунком в цвете `--azure`.
*   **Состояние**: При активности (`.show`) область расширяется до `max-height: 500px` и активирует тень `box-shadow` для визуального отделения от фона.
*   **Анимация**: Синхронизирована с родительской панелью (выезд из-под кнопки).
