# [Анимация появления карточек (_fade-up.scss)](../../../../../../../Frontend/Web/ground/branches/items/_items/animations/_fade-up.scss)

## Назначение
Определяет CSS-`@keyframes fadeUp` — анимацию «вылета снизу» для карточек предметов при подгрузке новых порций (бесконечный скролл). Запускается вручную из JS.

## Связи (Dependencies)
*   [Список всех предметов (ItemsBranch)](../../ItemsBranch.md): рассчитывает `delay` для каскадного появления и навешивает анимацию.

## Задаваемые стили

### `@keyframes fadeUp`
*   `from`: `opacity: 0; transform: translateY(30px)` — карточка прозрачна и смещена на 30px вниз.
*   `to`: `opacity: 1; transform: translateY(0)` — проявляется и встаёт на место.

## AI-контекст
*   Сам кейфрейм длительности/задержки не задаёт — их выставляет JS на элементе (`animation`/`animation-delay`). Это сделано ради «ручного» каскадного запуска.
