# [Анимация вылета (_fade-up.scss)](../../../../../../../Frontend/Web/ground/branches/items/_items/animations/_fade-up.scss)

## Назначение
Описание кастомной анимации `fadeUp`, используемой для плавного появления карточек предметов при динамической подгрузке (Infinite Scroll).

## Эффект
Элемент плавно меняет прозрачность от 0 до 1, одновременно поднимаясь на 30px вверх. Это создает эффект естественного заполнения сетки контентом.
