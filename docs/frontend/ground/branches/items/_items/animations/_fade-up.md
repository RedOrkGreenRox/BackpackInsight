# [Анимация появления карточек (_fade-up.scss)](../../../../../../../Frontend/Web/ground/branches/items/_items/animations/_fade-up.scss)

## Назначение
Реализует специфическую анимацию «вылета» снизу для новых порций предметов при бесконечном скролле.

## Ключевая логика
Использует `transform: translateY` и `opacity`. Синхронизируется с задержками (`delay`), рассчитываемыми в [Списке всех предметов](../../ItemsBranch.md).
