# [Оркестратор Вики (ItemsManager.ts)](../../../../../../../Frontend/Web/ground/branches/items/_items/managers/ItemsManager.ts)

## Назначение
Класс `ItemsManager` — это главный оркестратор Википедии предметов. Он объединяет логику работы поиска, фильтров, сохранения состояния и бесконечной прокрутки.

---

## Ключевая логика
*   Инициализирует состояние фильтрации из `sessionStorage` через `ItemsStateManager`.
*   Привязывает все обработчики DOM-событий к кнопкам, инпутам и выпадающим спискам.
*   Координирует бесконечный скролл с поэтапной догрузкой порций карточек предметов.
