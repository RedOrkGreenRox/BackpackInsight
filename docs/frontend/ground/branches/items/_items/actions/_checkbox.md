# [Стили чекбоксов (_checkbox.scss)](../../../../../../../Frontend/Web/ground/branches/items/_items/actions/_checkbox.scss)

## Назначение
Кастомный дизайн для чекбоксов фильтрации (например, «Только покупаемые»).

## Ключевая логика
Скрывает стандартный `input type="checkbox"` и рисует стилизованный квадрат в эстетике **Backpack Brawl**, используя псевдоэлементы `:before` и `:after` для галочки.
