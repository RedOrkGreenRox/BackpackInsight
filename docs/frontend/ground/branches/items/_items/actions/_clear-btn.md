# [Кнопка очистки фильтров (_clear-btn.scss)](../../../../../../../Frontend/Web/ground/branches/items/_items/actions/_clear-btn.scss)

## Назначение
Дизайн кнопки «Сбросить все», которая появляется при активных фильтрах.

## Ключевая логика
Акцентная кнопка с красными тонами или иконкой крестика. Использует плавную анимацию при наведении.
