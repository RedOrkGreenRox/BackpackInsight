# [Макет Вики-раздела (_layout.scss)](../../../../../../../Frontend/Web/ground/branches/items/_items/layout/_layout.scss)

## Назначение
Описание структуры верхнего уровня для страницы библиотеки предметов.

---

## Ключевые элементы

### `.wiki-section`
Основной контейнер раздела. Использует `clamp(80px, 15vh, 120px)` для верхнего отступа, чтобы заголовок не перекрывался фиксированной навигацией Shell на разных устройствах.

### `.wiki-subtitle`
Стилизация подзаголовка страницы. Использует приглушенный белый цвет (`opacity: 0.7`) для создания визуальной иерархии под главным названием.
