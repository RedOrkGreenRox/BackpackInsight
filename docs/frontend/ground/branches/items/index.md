# [Библиотека предметов (ItemsBranch.ts)](../../../../../Frontend/Web/ground/branches/items/ItemsBranch.ts)

## Назначение
Интерактивный справочник всех предметов игры. Реализует мощную систему поиска и фильтрации.

---

## Ключевая логика

### 1. Поиск (Fuse.js)
Использует библиотеку нечеткого поиска для нахождения предметов даже при наличии опечаток. Анализирует не только названия, но и скрытые теги предметов.

### 2. Многоуровневые фильтры
Поддерживает фильтрацию по:
*   Редкости.
*   Типу предмета (Weapon, Food, etc).
*   Связанному герою.

---

## AI-контекст
*   Сетка предметов рендерится виртуально: на больших списках отображаются только видимые карточки для экономии памяти.
*   Стили фильтров вынесены в модульные `.scss` файлы для предотвращения конфликтов.
