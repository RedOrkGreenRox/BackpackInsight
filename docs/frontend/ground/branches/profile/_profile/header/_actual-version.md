# [Индикация версии игры (_actual-version.scss)](../../../../../../../Frontend/Web/ground/branches/profile/_profile/header/_actual-version.scss)

## Назначение
Стилизация технической метки в углу заголовка профиля, которая показывает версию игрового клиента.

## Характеристики
*   **Типографика**: Крайне мелкий шрифт (`0.3em`) с низким весом (300).
*   **Видимость**: Почти прозрачный белый цвет (`rgba(..., 0.15)`), чтобы не отвлекать от основной статистики.
*   **Позиционирование**: Абсолютно прижато к правому нижнему углу заголовка (`2%` отступ).
