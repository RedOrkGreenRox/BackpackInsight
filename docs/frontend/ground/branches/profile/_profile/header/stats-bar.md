# [Рендерер панели ресурсов (StatsBarRenderer)](../../../../../../../Frontend/Web/ground/branches/profile/_profile/header/stats-bar.ts)

## Назначение
Этот компонент отвечает за визуализацию игровых ресурсов и прогресса: уровня, опыта (XP), трофеев, гемов и монет. Это «пульт управления» статусом игрока в **Backpack Brawl**.

## Связи (Dependencies)
*   Вызывается из основного [Рендерера заголовка](header.md).
*   Берет актуальные значения из [Интерфейсов профиля](../utils/profile-types.md).
*   **Изображения**: Использует иконки ресурсов из папки `/images/profile/`.

---

## Подробное описание методов

### `static render(data: ProfileData)`
Формирует горизонтальную сетку карточек ресурсов.
*   **XP Progress**: Отображает дробное значение "Текущий опыт / Нужно до уровня". Использует локальное форматирование чисел (`toLocaleString`) для добавления разделителей тысяч.
*   **Trophies**: Суммирует базовые трофеи и бонусные трофеи игрока для отображения итогового рейтинга.
*   **Форматы**: Каждая иконка обернута в `<picture>` с поддержкой AVIF и WebP, что обеспечивает минимальный вес страницы при сохранении качества графики.

## AI-контекст
*   **Локализация чисел**: Метод `fmt` внутри рендерера гарантирует, что большие числа (например, 1,000,000 монет) будут выглядеть аккуратно на любом языке.
*   **Дизайн**: Каждая карточка имеет класс `.stat-player-card`, который задает полупрозрачный фон и скругление углов в соответствии со стилем игры.
