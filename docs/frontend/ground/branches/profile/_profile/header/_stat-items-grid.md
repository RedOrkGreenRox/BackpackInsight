# [Сводка предметов по редкости (_stat-items-grid.scss)](../../../../../../../Frontend/Web/ground/branches/profile/_profile/header/_stat-items-grid.scss)

## Назначение
Стилизация панели, отображающей суммарное количество предметов каждой редкости в инвентаре игрока.

---

## Ключевые элементы

### Контейнер (`.stats-items-grid`)
*   **Стиль**: Компактная плашка с темным фоном и скруглением 10px.
*   **Hover**: Реализован "всплывающий" эффект с акцентным свечением.

### Элемент редкости (`.rarity-item`)
*   **Иконка**: Использует `clamp(18px, 4vw, 24px)` для адаптивного размера иконки-карточки.
*   **Счетчик**: Крупный жирный шрифт (700) бежевого цвета.

---

## Адаптивность
На мобильных устройствах элементы сводки перестраиваются в несколько рядов с уменьшенными зазорами (4px), чтобы поместиться в ширину экрана.
