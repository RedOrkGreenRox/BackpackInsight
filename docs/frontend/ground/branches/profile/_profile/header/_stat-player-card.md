# [Карточка ресурса игрока (_stat-player-card.scss)](../../../../../../../Frontend/Web/ground/branches/profile/_profile/header/_stat-player-card.scss)

## Назначение
Стилизация отдельной пары "Иконка + Значение" для ресурсов (Золото, Гемы, XP). Используется внутри общей сетки статистики.

## Характеристики
*   **Иконка**: Масштабируется от 20px до 32px.
*   **Текст**: Жирный белый шрифт с адаптивным размером (14-20px), предотвращающим разрывы длинных чисел.
