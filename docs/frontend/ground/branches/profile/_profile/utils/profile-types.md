# [Типы данных профиля (profile-types.ts)](../../../../../../../Frontend/Web/ground/branches/profile/_profile/utils/profile-types.ts)

## Назначение
Центральный файл TypeScript-интерфейсов для страницы профиля. Он обеспечивает строгую типизацию данных, получаемых от API и передаваемых между компонентами.

---

## Ключевые интерфейсы

### 1. `ProfileData`
Главный объект, описывающий полный снимок игрока.
*   **Ресурсы**: трофеи, гемы, монеты.
*   **Прогресс**: текущий опыт и опыт до уровня.
*   **Коллекции**: массивы объектов `Hero` и `Item`.

### 2. `Hero`
Параметры персонажа, включая лигу, рейтинг и статус престижа.

### 3. `Item`
Атомарные данные предмета в инвентаре игрока.
