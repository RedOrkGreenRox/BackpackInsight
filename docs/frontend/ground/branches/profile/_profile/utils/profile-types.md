## Назначение
Этот файл содержит TypeScript-интерфейсы, описывающие структуру данных профиля игрока **Backpack Brawl**. Это локальное расширение глобальных типов, специфичное для логики страницы профиля.

## Связи (Dependencies)
*   `[ProfileStateManager.ts](../managers/ProfileStateManager.md)`: Реэкспортирует тип `SavedState` для управления сохраненным состоянием UI.
*   `[global.d.ts](../../../../types/global.md)`: Является основой для этих интерфейсов.

---

## Описание интерфейсов

### `Hero`
Описывает данные конкретного героя в профиле:
*   `name`: Имя героя.
*   `level`: Текущий уровень.
*   `rating`: Рейтинг героя (Trophy).
*   `experience` / `exp_req`: Текущий и требуемый опыт для следующего уровня.
*   `prestige`: Флаг наличия престижа (особая рамка).
*   `league`: Название лиги (Bronze, Gold и т.д.).
*   `skin_num`: ID выбранного скина.

### `Item`
Описывает предмет в коллекции игрока:
*   `name`: Название.
*   `rarity`: Редкость (влияет на сортировку и цвет).
*   `level`: Текущий уровень предмета.
*   `cards` / `cards_need`: Прогресс сбора карточек для улучшения.

### `ProfileData`
Главный интерфейс данных страницы:
*   Агрегирует массивы `heroes` и `items`.
*   Содержит метаданные: `nickname`, `trophy`, `gems`, `coins`, `area`.
*   `profile_skins`: Словарь доступных скинов для каждого героя.
*   `itemsSort`: Состояние сортировки предметов в UI ('rarity' | 'level').
