# [Frontend/Web/ground/branches/profile/_profile/utils/rarity-weights.ts](../../../../../../../Frontend/Web/ground/branches/profile/_profile/utils/rarity-weights.ts)

## Назначение
Этот файл определяет числовые веса (приоритеты) для редкостей предметов в **Backpack Brawl**. Эти веса используются алгоритмами сортировки для того, чтобы "Unique" предметы всегда были выше "Common".

## Связи (Dependencies)
*   `[SortController.ts](../sort/SortController.md)`: Использует эти веса для вычисления порядка элементов в гриде предметов.

---

## Константы

### `rarityWeights`
Объект-словарь, где ключ — это строка редкости, а значение — числовой приоритет.

| Редкость | Вес |
| :--- | :--- |
| **Unique** | 100 |
| **Boon** | 90 |
| **Relic** | 80 |
| **Mythic** | 70 |
| **Legendary** | 60 |
| **Epic** | 50 |
| **Rare** | 40 |
| **Common** | 30 |
| **Special** | 20 |

## AI-контекст
*   **Логика сортировки**: Если нужно инвертировать порядок редкостей или добавить новую редкость (например, "Event"), правки вносятся именно в этот словарь.
*   **Консистентность**: Веса должны соответствовать игровой ценности предметов в **Backpack Brawl**.
