# [Секция героев (heroes-section.ts)](../../../../../../../Frontend/Web/ground/branches/profile/_profile/heroes/heroes-section.ts)

## Назначение
Рендерер контейнера для списка всех персонажей игрока. Обеспечивает структуру сетки и панель управления сортировкой.

---

## Ключевая логика

### 1. Первичная сортировка
Перед рендерингом герои сортируются по эталонному алгоритму:
1.  **Уровень + Престиж**: (Престижные герои выше).
2.  **Рейтинг**.
3.  **Имя**.
Это гарантирует консистентность вида страницы до начала взаимодействия пользователя с кнопками сортировки.

### 2. Панель управления
Отрисовывает две кнопки:
*   `#sortToggle`: Смена режима (Уровень / Рейтинг).
*   `#invertToggle`: Смена направления (По возрастанию / По убыванию).

---

## Связи (Dependencies)
*   Использует [`HeroCardRenderer`](hero-card.md) для отрисовки каждого персонажа.
