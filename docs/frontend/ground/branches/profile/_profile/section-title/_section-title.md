# [Заголовок секции профиля (_section-title.scss)](../../../../../../../Frontend/Web/ground/branches/profile/_profile/section-title/_section-title.scss)

## Назначение
Стили заголовков секций `.section-title` (например, «Герои», «Предметы») на странице профиля.

## Задаваемые стили
### `.section-title`
*   `color: var(--text-default-color); font-size:2rem; margin-bottom:20px; text-align:center; text-shadow:0 2px 4px rgba(0,0,0,0.8)`.
*   `opacity:1` — принудительно видимый (глобальный сброс скрывает `h2`).
*   `&.loaded { opacity:1 }` — совместимость с анимацией появления.

## AI-контекст
*   `opacity:1` тут не случаен: глобальный reset прячет заголовки до анимации; этот класс гарантирует видимость секционных заголовков.
