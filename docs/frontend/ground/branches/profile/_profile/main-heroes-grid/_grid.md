# [Сетка героев профиля (_grid.scss)](../../../../../../../Frontend/Web/ground/branches/profile/_profile/main-heroes-grid/_grid.scss)

## Назначение
Отвечает за расположение карточек героев в основном блоке профиля.

## Связи (Dependencies)
*   [Глобальные переменные](../../../../roots/_roots/_vars.md): Зазоры (gap) между карточками.
*   Используется [Рендерером секции героев](../heroes/heroes-section.md).

## Ключевая логика
Использует `display: grid`.
*   **Гибкость**: Автоматически рассчитывает количество колонок (от 1 на мобилках до 3-4 на десктопах) с помощью `grid-template-columns: repeat(auto-fill, minmax(...))`.
*   **Центрирование**: Если героев мало, они выравниваются по центру.

## AI-контекст
Этот файл критичен для работы [Контроллера сортировки](../sort/SortController.md). Плавные перемещения карточек при сортировке зависят от свойств этого грида.
