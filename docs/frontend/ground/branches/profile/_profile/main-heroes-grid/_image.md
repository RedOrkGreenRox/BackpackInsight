# [Изображение героя (_image.scss)](../../../../../../../Frontend/Web/ground/roots/_roots/items/_item-image.scss)

## Назначение
Управление отображением большого портрета персонажа и логикой смены скинов.

---

## Особенности
*   **Размер**: Фиксированный 160x160px (уменьшается до 90px на мобильных).
*   **Тень**: Используется `drop-shadow` для создания выраженного силуэта поверх фона карточки.
*   **Анимация смены скина**: При добавлении класса `.changing-skin` (через JS) изображение плавно исчезает и уменьшается в масштабе, создавая эффект "переворачивания" или быстрой замены.
