# [Карточка героя (_card.scss)](../../../../../../../Frontend/Web/ground/branches/profile/_profile/main-heroes-grid/_card.scss)

## Назначение
Определяет детальный дизайн большой карточки героя, включая свечение рамок престижа и кнопки смены скинов.

## Связи (Dependencies)
*   [Глобальные переменные](../../../../roots/_roots/_vars.md): Цвета рамок лиг.
*   Используется [Рендерером карточки героя](../heroes/hero-card.md).

## Ключевая логика
*   **Ховер-эффекты**: При наведении карточка слегка приподнимается и увеличивает яркость.
*   **Кнопки скинов**: Абсолютно позиционированные стрелки по бокам карточки, которые появляются только при наличии нескольких скинов.
*   **Кнопки скинов**: `.skin-btn` (и `.prev-skin`/`.next-skin`) — абсолютно позиционированные по бокам зоны переключатели скинов; сужаются на мобильных (`width` 40px → 30px → 25px).

## AI-контекст
Изображения героев внутри карточек имеют `transition: opacity`, что позволяет [Менеджеру скинов](../managers/ProfileSkinsManager.md) плавно менять их без мерцания.
