# [Уровень и Рейтинг (_level-rating.scss)](../../../../../../../Frontend/Web/ground/branches/profile/_profile/main-heroes-grid/_level-rating.scss)

## Назначение
Описание графических индикаторов уровня и лиги в большой карточке персонажа.

## Характеристики
*   **Иерархия**: Слой `z-index: 1` для графических рамок и `z-index: 2` для числовых значений.
*   **Типографика**: Исключительно жирный шрифт (800) с двойной обводкой тенью (`text-shadow`) для читаемости на фоне сложных ассетов Unity.
