# [Верхний ряд карточки (_header-row.scss)](../../../../../../../Frontend/Web/ground/branches/profile/_profile/main-heroes-grid/_header-row.scss)

## Назначение
Стилизация верхней информационной плашки внутри большой карточки героя.

## Характеристики
*   **Дизайн**: Имеет собственный полупрозрачный фон (`white 0.05`), создающий эффект вложенного слоя.
*   **Макет**: Flex-блок с распределением `space-between` для разнесения уровня и рейтинга.
*   **Скругление**: 15px.
