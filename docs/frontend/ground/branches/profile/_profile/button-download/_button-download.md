# [Стиль кнопки скачивания (_button-download.scss)](../../../../../../../Frontend/Web/ground/branches/profile/_profile/button-download/_button-download.scss)

## Назначение
Стилизация кнопки "Сохранить карточку" в профиле игрока. Файл реализует Glass UI стиль с акцентным свечением при взаимодействии.

---

## Визуальные характеристики
*   **Стиль**: "Стеклянный" интерфейс (`background: rgba(0, 0, 0, 0.5)`) с полупрозрачной рамкой.
*   **Типографика**: Шрифт `Signika`, жирный (700), все буквы заглавные.
*   **Форма**: Формат "пилюли" (`border-radius: 50px`).

## Интерактивность
*   **Hover**: Кнопка смещается на 2px вниз, а вокруг неё появляется интенсивное голубое свечение (`--azure`).
*   **Active**: Усиливается смещение (4px) и уменьшается яркость (`brightness(0.9)`), имитируя физическое нажатие.

---

## Адаптивность
На мобильных устройствах (`max-width: 768px`) кнопка растягивается на всю ширину контейнера для удобства нажатия пальцем.
