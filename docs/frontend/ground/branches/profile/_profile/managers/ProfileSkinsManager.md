# [Менеджер скинов (ProfileSkinsManager.ts)](../../../../../../../Frontend/Web/ground/branches/profile/_profile/managers/ProfileSkinsManager.ts)

## Назначение
Логический модуль для управления косметическими обликами героев. Он обрабатывает список разблокированных предметов игрока и сопоставляет их с доступными изображениями на сервере.

---

## Функционал

### 1. Парсинг данных
Извлекает информацию о скинах из технического блока профиля. Приводит имена героев к нижнему регистру для предотвращения ошибок маппинга.

### 2. Сбор доступных обликов
Метод `getUniqueSkins` гарантирует:
*   Скин `01` (базовый) всегда доступен любому герою.
*   Отсутствие дубликатов в списке выбора.
*   Алфавитную сортировку кодов скинов.

### 3. Генерация путей к изображениям
Автоматически строит URL-адреса для иконок героев в форматах **WebP** и **AVIF** на основе имени персонажа и кода скина.

---

## AI-контекст
*   Этот менеджер изолирует логику именования ассетов от UI-компонентов. Если структура папок с изображениями героев изменится, правки потребуются только здесь.
