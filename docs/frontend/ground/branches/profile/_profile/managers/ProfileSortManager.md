# [Логика сортировки (ProfileSortManager.ts)](../../../../../../../Frontend/Web/ground/branches/profile/_profile/managers/ProfileSortManager.ts)

## Назначение
Инкапсулирует чистые алгоритмы сортировки коллекции предметов.

---

## Алгоритм («Стабильная сортировка»)

Функция учитывает 4 уровня приоритетов:
1.  **Тип сортировки**: Если выбрано `rarity`, сначала сравниваются веса из справочника.
2.  **Уровень**: Предметы с более высоким LVL всегда выше при прочих равных.
3.  **Процент прокачки**: Если уровни равны, выше оказывается тот предмет, у которого больше карточек в процентах до следующего уровня.
4.  **Имя**: Финальное выравнивание по алфавиту.

---

## AI-контекст
*   Сортировка выполняется на **копии** массива (`[...items]`), что гарантирует иммутабельность исходных данных профиля.
