# [Менеджер скриншотов (screenshot-manager.ts)](../../../../../../../Frontend/Web/ground/branches/profile/_profile/managers/screenshot-manager.ts)

## Назначение
Класс `ScreenshotManager` целиком отвечает за сохранение шапки профиля как PNG-изображения. Он сам выполняет захват DOM через `html2canvas` (отдельной «утилиты скриншотов» в проекте нет — вся логика здесь).

## Связи (Dependencies)
*   **`html2canvas`** (npm): подключается **лениво** через динамический `import('html2canvas')` — библиотека не попадает в основной бандл и грузится только при первом наведении/использовании.
*   `[Система i18n](../../../../localization/i18n.md)`: в конструктор передаётся функция перевода `t`, используемая для подписей кнопки (`screenshot_saving`, `screenshot_error`).
*   **DOM**: работает с `.profile-header` (захватываемая область) и кнопкой `#saveProfileBtn`.

## Подробное описание (реальная реализация)

### Конструктор `(container: HTMLElement, t: (key: string) => string)`
*   Сохраняет контейнер и функцию перевода. Готовит массив `cleanupFns` и кэш промиса `html2canvasPromise`.

### `init()`
*   Вешает на `#saveProfileBtn` три слушателя: `click` → `takeScreenshot()`, а `pointerenter`/`focus` → `preloadHtml2Canvas()` (предзагрузка библиотеки до клика).
*   Все слушатели регистрируются с деинсталляторами в `cleanupFns`.

### `preloadHtml2Canvas(): Promise<any>`
*   Лениво и единожды (`??=`) импортирует `html2canvas`, кэшируя промис, чтобы повторные вызовы не запускали загрузку снова.

### `takeScreenshot()`
1.  Находит `.profile-header`; если нет — выходит.
2.  Переводит кнопку в состояние «сохранение» (`t('screenshot_saving')`, `opacity 0.7`, `disabled`).
3.  Дожидается `html2canvas`, **временно** форсирует фиксированный размер области (900×600, `scale(1)`, `overflow:hidden`) ради стабильного кадра.
4.  Рендерит canvas с опциями `useCORS`, `allowTaint`, `scale: 2`, фиксированными `width/height` и расширенным `windowWidth/windowHeight`.
5.  Восстанавливает исходные inline-стили, создаёт временную `<a download>` с `canvas.toDataURL('image/png')`, кликает и удаляет ссылку. Имя файла: `profile-screenshot-{timestamp}.png`.
6.  В `catch` логирует ошибку и показывает на кнопке `t('screenshot_error')`.

## AI-контекст
*   **CORS**: иконки предметов/арен должны быть доступны для canvas (флаги `useCORS`/`allowTaint`), иначе снимок выйдет пустым.
*   **Фиксированный кадр**: размеры 900×600 захардкожены ради единообразия «карточек игрока». Меняя вёрстку шапки, проверяйте, что она помещается в этот кадр.
