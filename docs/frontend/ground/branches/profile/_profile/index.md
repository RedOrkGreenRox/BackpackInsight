## Назначение
Этот файл является точкой входа (Barrel) для папки `_profile/`. Он экспортирует все ключевые менеджеры и рендереры, предоставляя чистый интерфейс для внешнего бранча [ProfileBranch](../../ProfileBranch.md).

---

## AI-контекст
*   Использование этого файла позволяет другим модулям импортировать всё необходимое из одной точки, предотвращая "ад относительных путей" (`../../../../...`).
*   Здесь также реэкспортируются типы данных, обеспечивая единый контракт на использование интерфейса профиля во всём приложении.
