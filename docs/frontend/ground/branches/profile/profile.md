# [Стили страницы профиля (profile.scss)](../../../../Frontend/Web/ground/branches/profile/profile.scss)

## Назначение
Корневой стилевой файл для всей ветки Профиля. Он не содержит конкретных стилей элементов, а работает как **точка сборки**, объединяя все мелкие компоненты через `@use`.

## Связи (Dependencies)
Импортирует всю архитектуру стилей профиля:
*   [Глобальные переменные](../../roots/_roots/_vars.md).
*   [Стили шапки](./_profile/header/_header.md).
*   [Сетку героев](./_profile/main-heroes-grid/_grid.md).
*   [Сетку предметов](./_profile/items/_items-grid.md).
*   [Кнопки сортировки](./_profile/buttons-sort/_buttons-sort.md).

## Ключевая логика
Файл обеспечивает изоляцию стилей страницы Профиля, предотвращая их утечку на Главную страницу. Все селекторы здесь начинаются с контейнера `.profile-header` или `.section`.

## AI-контекст
При добавлении нового визуального модуля в профиль (например, блока с ачивками), его `.scss` файл должен быть подключен именно здесь.
