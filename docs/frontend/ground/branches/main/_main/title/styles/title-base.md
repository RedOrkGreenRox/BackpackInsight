# [Базовый стиль заголовка (title-base.scss)](../../../../../../../../Frontend/Web/ground/branches/main/_main/title/styles/title-base.scss)

## Назначение
Детальное оформление главного заголовка страницы.

## Характеристики
*   **Типографика**: Размер `3em`, вес `800` (Extra Bold).
*   **Видимость**: Использует двойную тень (`text-shadow`): жесткую для контура и мягкую для свечения. Это критично для читаемости белого текста на динамическом фоне.
*   **Эффекты**: Незначительное увеличение (`scale(1.02)`) при наведении.
