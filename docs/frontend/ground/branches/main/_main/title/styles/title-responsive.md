# [Адаптивность заголовка (title-responsive.scss)](../../../../../../../../Frontend/Web/ground/branches/main/_main/title/styles/title-responsive.scss)

## Назначение
Медиа-переопределения размеров `.main-title`. База — в [title-base](title-base.md).

## Задаваемые стили

### `@media (max-width: 768px)`
*   `.main-title`: `font-size:2.2em; margin-bottom:20px`.

### `@media (max-width: 480px)`
*   `.main-title`: `font-size:1.8em; margin-bottom:15px; letter-spacing:0.5px`.

## AI-контекст
*   Только `@media`. Базовые цвет/тени/hover не дублируются.
