# [Рендерер заголовка (TitleRenderer.ts)](../../../../../../../Frontend/Web/ground/branches/main/_main/title/title.ts)

## Назначение
Отвечает за генерацию HTML-кода главного заголовка страницы.

## Особенности
*   **Локализация**: Принимает функцию `t`, обеспечивая корректный перевод заголовка.
*   **Анимация**: Содержит атрибут `data-aos="fade-down"`, что заставляет заголовок плавно "выезжать" сверху при загрузке.
