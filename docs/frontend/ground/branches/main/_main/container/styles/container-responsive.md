# [Адаптивность контейнера (container-responsive.scss)](../../../../../../../../Frontend/Web/ground/branches/main/_main/container/styles/container-responsive.scss)

## Назначение
Корректирует размеры контейнера под мобильные устройства и планшеты.

## Характеристики
*   **Mobile**: При ширине `< 768px` контейнер расширяется до `98%` экрана, а верхний отступ увеличивается до `15vh` для компенсации системных элементов управления.
*   **Landscape**: При малой высоте экрана отступы минимизируются до `5vh`.
