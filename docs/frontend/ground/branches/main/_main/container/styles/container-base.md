# [Базовый макет контейнера (container-base.scss)](../../../../../../../../Frontend/Web/ground/branches/main/_main/container/styles/container-base.scss)

## Назначение
Определяет фундаментальные геометрические параметры главного контейнера страницы.

## Характеристики
*   **Ширина**: Ограничена `900px` для сохранения читаемости контента.
*   **Центрирование**: `margin: 0 auto`.
*   **Слои**: `z-index: 10` — гарантирует, что контент находится поверх эффектов фона.
*   **Отступы**: Использует `clamp(20px, 5vh, 60px)` для адаптивного вертикального пространства.
