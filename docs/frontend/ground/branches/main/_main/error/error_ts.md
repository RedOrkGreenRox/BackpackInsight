# [Рендерер контейнера ошибок (error.ts)](../../../../../../../Frontend/Web/ground/branches/main/_main/error/error.ts)

## Назначение
Класс `ErrorRenderer` отвечает за генерацию HTML-скелета для вывода уведомлений об ошибках на главной странице.

## Функционал
*   Генерирует скрытый `div` с ID `errorContainer`.
*   Добавляет ARIA-атрибуты для доступности (Accessibility), чтобы ошибки озвучивались скринридерами.
