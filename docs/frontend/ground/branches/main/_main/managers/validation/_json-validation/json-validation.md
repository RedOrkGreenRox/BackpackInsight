# [Стили редактора ошибок (json-validation.scss)](../../../../../../../../../Frontend/Web/ground/branches/main/_main/managers/validation/_json-validation/json-validation.scss)

## Назначение
Стилизация "мини-редактора" кода, который появляется при обнаружении синтаксических ошибок в вставленном JSON-логе.

---

## Ключевые элементы

### `.json-validation-error`
*   **Фон**: Почти прозрачный белый (`rgba(255, 255, 255, 0.05)`).
*   **Типографика**: Моноширинный шрифт (`Courier New`) для имитации среды разработки.
*   **Макет**: Flex-блок с горизонтальным разделением (номера строк | код).

### Подсветка ошибок
*   **`.error-line-numbers`**: Панель номеров строк с выделением ошибочного индекса красным цветом.
*   **`.error-char`**: Ярко-красный фон для конкретного символа, на котором «сломался» парсер.
*   **`.code-line.error-line`**: Мягкая красная подсветка всей строки с ошибкой.

---

## Оптимизация
*   В режиме [**Low-Res Mode**](../../../../../../roots/_roots/_low-res.md) фон и подсветка становятся более контрастными за счет увеличения непрозрачности, чтобы текст оставался читаемым без эффектов размытия.
