# [Менеджер форм (FormManager.ts)](../../../../../../../Frontend/Web/ground/branches/main/_main/managers/FormManager.ts)

## Назначение
Простой контроллер для перехвата стандартного поведения HTML-формы. Гарантирует, что данные будут отправлены через SPA-логику, а не через перезагрузку страницы.

## Функционал
*   Подписка на событие `submit`.
*   Вызов `preventDefault()` для блокировки стандартной отправки.
*   Передача управления в колбэк `onSubmit`, который связан с [SubmitManager](submit/SubmitManager.md).
