# [Базовый стиль подсказки (upload-hint-base.scss)](../../../../../../../../Frontend/Web/ground/branches/main/_main/upload-zone/hint-styles/upload-hint-base.scss)

## Назначение
Стилизация фонового текста внутри зоны загрузки, который инструктирует пользователя.

## Характеристики
*   **Типографика**: Крупный текст (`1.5rem`), полужирный.
*   **Иконки**: Размер `3rem` с эффектом падающей тени.
*   **Слой**: `z-index: 1` — находится под курсором и текстом `textarea`.
