# [Подсказка для ПК (upload-hint-pc-only.scss)](../../../../../../../../Frontend/Web/ground/branches/main/_main/upload-zone/hint-styles/upload-hint-pc-only.scss)

## Назначение
Управляет видимостью элемента `.upload-hint .pc-only` — части подсказки, актуальной только для десктопа (например, упоминание Drag-and-Drop, которого нет на тач-устройствах).

## Задаваемые стили

### `.upload-hint .pc-only`
*   По умолчанию: `display: block` (виден).
*   `@media (max-width: 1024px)`: `display: none` — скрывается на планшетах/мобильных.

## AI-контекст
*   Граница 1024px намеренно шире 768px: на планшетах DnD тоже ненадёжен, поэтому подсказку прячут раньше «мобильного» брейкпоинта.
