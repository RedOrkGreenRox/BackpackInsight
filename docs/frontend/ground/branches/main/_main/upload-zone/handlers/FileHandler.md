# [Frontend/Web/ground/branches/main/_main/upload-zone/handlers/FileHandler.ts](../../../../../../../../Frontend/Web/ground/branches/main/_main/upload-zone/handlers/FileHandler.ts)

## Назначение
Низкоуровневый обработчик файловых операций в зоне загрузки. Его задача — перехватить файл, выбранный через диалоговое окно системы или перетащенный мышью, и превратить его в текстовую строку для дальнейшего анализа.

## Связи (Dependencies)
*   `[JsonValidator.ts](../../managers/validation/JsonValidator.md)`: Передает прочитанный текст валидатору для проверки структуры.
*   `[UIHandler.ts](./UIHandler.md)`: Уведомляет визуальный интерфейс о том, что файл начал или закончил обрабатываться (для показа спиннеров).
*   `[LoadingStates.ts](../../../../../../utils/LoadingStates.md)`: Использует глобальные состояния загрузки для блокировки кнопки выбора файла.

## Ключевая логика
Использует `FileReader API` для асинхронного чтения содержимого.

### Процесс:
1.  **Перехват**: Получает объект `File`.
2.  **Валидация типа**: Проверяет, что это текстовый файл или `.json`.
3.  **Чтение**: Вызывает `readAsText()`.
4.  **Цепочка обработки**: Если чтение успешно, данные передаются в `[JsonValidator.ts](../../managers/validation/JsonValidator.md)`. Если нет — вызывается обработка ошибок через `[eventUtils.ts](../../../../../../utils/eventUtils.md)`.

## AI-контекст
*   **Ограничения**: В этом файле стоит жесткая проверка на размер файла (обычно лог профиля **Backpack Brawl** весит не более пары мегабайт).
*   **Исключения**: При возникновении ошибок чтения (например, файл занят другой программой), `FileHandler` должен корректно сбросить состояние `[UIHandler.md](./UIHandler.md)`, чтобы пользователь мог попробовать еще раз.
