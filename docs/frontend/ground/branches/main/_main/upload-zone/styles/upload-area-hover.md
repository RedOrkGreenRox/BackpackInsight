# [Индикация Drag-over (upload-area-hover.scss)](../../../../../../../../Frontend/Web/ground/branches/main/_main/upload-zone/styles/upload-area-hover.scss)

## Назначение
Описание визуального отклика при перетаскивании файла на область загрузки.

## Характеристики
*   **Действие**: При добавлении класса `.drag-over` рамка становится сплошной и ярко-белой, а фон значительно темнеет, создавая визуальный "ловушку" для файла.
