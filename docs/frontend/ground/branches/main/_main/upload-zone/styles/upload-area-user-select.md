# [Выделение и перетаскивание зоны (upload-area-user-select.scss)](../../../../../../../../Frontend/Web/ground/branches/main/_main/upload-zone/styles/upload-area-user-select.scss)

## Назначение
Разрешает выделение текста и перетаскивание элементов внутри `.upload-area` (важно для вставки/копирования JSON и DnD).

## Задаваемые стили

### `.upload-area, .upload-area *`
*   `user-select: text` (+ вендорные `-webkit-/-moz-/-ms-`) — текст можно выделять во всех элементах зоны.
*   `-webkit-user-drag: element` — разрешает перетаскивание элементов (WebKit).

## AI-контекст
*   Намеренное переопределение: остальной UI часто запрещает выделение (`user-select:none`), но зона ввода JSON обязана его разрешать.
