# [Локальные анимации главной (animations.scss)](../../../../../../../Frontend/Web/ground/branches/main/_main/animations/animations.scss)

## Назначение
Специфические анимации только для главной страницы: плавное появление текстового поля и эффекты зоны загрузки.

## Связи (Dependencies)
*   [Глобальные анимации](../../../../roots/_roots/_animations.md): Расширяет базовые эффекты.

## Ключевая логика
*   `drop-hint-anim`: Анимация "подпрыгивания" иконки при наведении файла на зону загрузки.
*   `fade-slide-up`: Эффект появления кнопок анализа.

## AI-контекст
Эти анимации используются [Интерфейсом загрузки](../upload-zone/handlers/UIHandler.md) для обратной связи с пользователем.
