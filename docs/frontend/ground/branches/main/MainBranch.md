# [Главная страница (MainBranch.ts)](../../../../../Frontend/Web/ground/branches/main/MainBranch.ts)

## Назначение
Главный класс «Бранча» для стартовой страницы приложения. Он координирует работу зоны загрузки профиля и первичного взаимодействия пользователя с системой.

## Связи (Dependencies)
*   **Стили**: Ссылается на [Корневые стили главной](./main.md).
*   **Родитель**: Наследуется от [BaseBranch](../BaseBranch.md).
*   **Управление**: Инициализирует [Оркестратор главной](./_main/managers/MainManager.md).

## Ключевая логика
Бранч инициализирует `MainManager`, который, в свою очередь, собирает все части страницы воедино.

### Методы:
*   `init()`: Проверяет наличие черновиков через менеджеры и настраивает UI.
*   `render()`: Отрисовывает структуру главной страницы: приветствие, инструкции и зону загрузки.

## AI-контекст
*   **Точка входа**: Это «лицо» приложения. Любое изменение в процессе первого знакомства пользователя с продуктом начинается здесь.
*   **Пересмотр чанков**: Поскольку это первая страница, её код должен быть максимально легким. Все тяжелые менеджеры должны грузиться оптимально.
