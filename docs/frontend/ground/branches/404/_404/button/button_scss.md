# [Кнопка 404: стили (button.scss)](../../../../../../../Frontend/Web/ground/branches/404/_404/button/button.scss)

## Назначение
Стилизация интерактивной кнопки на странице ошибки.

## Характеристики
*   **Дизайн**: Темный Glass UI с белой рамкой.
*   **Hover**: Реализован эффект "вспышки": рамка становится чисто белой, и добавляется интенсивное внешнее свечение.
*   **Тень**: Использует `box-shadow` для создания объема над затемненным фоном.
