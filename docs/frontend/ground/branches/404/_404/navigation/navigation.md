# [Навигация 404 (navigation.ts)](../../../../../../../Frontend/Web/ground/branches/404/_404/navigation/navigation.ts)

## Назначение
Класс `NavigationManager` навешивает обработчик клика на кнопку «домой» страницы 404 и возвращает функцию-деструктор для его снятия.

## Связи (Dependencies)
*   [Генератор UI / роутер (Gen.ts)](../../../../roots/Gen.md): по клику вызывает `Gen.getInstance().navigate('/')`.
*   [Рендерер кнопки (button.ts)](../button/button.md): создаёт кнопку с `id="homeBtn"`, которую ищет этот менеджер.

## Подробное описание
*   `static initNavigation(container): () => void`
    *   Находит `#homeBtn` в контейнере; если нет — возвращает «пустой» деструктор.
    *   Вешает `click` → SPA-навигация на `/`.
    *   Возвращает функцию, снимающую слушатель (вызывается в `destroy()` бранча).

## AI-контекст
*   Контракт по `id="homeBtn"` синхронизирован с `ButtonRenderer`. Возврат деструктора — обязательный паттерн против утечек слушателей.
