# [Тело страницы 404 (body.scss)](../../../../../../../Frontend/Web/ground/branches/404/_404/body/body.scss)

## Назначение
Базовый класс `.body-404`, описывающий поведение всего тела страницы ошибки. Навешивается на `<body>` через `@extend` из [корневого 404.scss](../../404.md).

## Задаваемые стили

### `.body-404`
*   `position: relative` — система координат для `position: absolute` у дочерних элементов (фон, оверлей).
*   `min-height: 100dvh` — высота на весь динамический вьюпорт (корректно учитывает адресную строку на мобильных).
*   `margin: 0` — сброс внешних отступов браузера.
*   `overflow: hidden` — блокирует прокрутку (страница-«заглушка» без скролла).
*   `text-align: center` — центрирование текстового контента.
*   `background-color: #000` — запасной фон на время загрузки фоновой картинки.

## AI-контекст
*   Используется `dvh` вместо `vh` — намеренно, ради корректной высоты на мобильных браузерах.
