# [Фон 404: стили (background.scss)](../../../../../../../Frontend/Web/ground/branches/404/_404/background/background.scss)

## Назначение
Стилизация полноэкранного фонового изображения на странице ошибки.

## Характеристики
*   **Позиционирование**: Фиксированное (`fixed`), занимает 100% вьюпорта.
*   **Слой**: Самый нижний (`z-index: -2`), находится под контентом и оверлеем.
*   **Оптимизация**: Использует `object-fit: cover` для правильного масштабирования картинки без искажения пропорций.
