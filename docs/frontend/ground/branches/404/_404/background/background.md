# [Фон страницы 404 (background.scss)](../../../../../../../Frontend/Web/ground/branches/404/_404/background/background.scss)

## Назначение
Стили для анимированного или декоративного фона страницы ошибки. Создает визуальный контекст «пустоты» или «потерянного рюкзака».

## Связи (Dependencies)
*   [Переменные дизайна](../../../../../roots/_roots/_vars.md): Цвета для градиентов фона.
*   Используется в [NotFoundBranch](../../NotFoundBranch.md).

## Ключевая логика
Задает `position: fixed` и `z-index: -1`, чтобы фон всегда заполнял весь экран независимо от размера окна браузера.
