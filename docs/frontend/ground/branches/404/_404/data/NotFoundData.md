# [Данные 404 (NotFoundData.ts)](../../../../../../../Frontend/Web/ground/branches/404/_404/data/NotFoundData.ts)

## Назначение

`NotFoundData` — data-модуль для страницы 404 в рамках [`StructuredBranch`](../../../../roots/StructuredBranch.md). Страница 404 не требует загрузки внешних данных, поэтому модуль возвращает пустой контекст.

## Связи

*   Вызывается из [`NotFoundBranch`](../../NotFoundBranch.md), наследующего [`StructuredBranch`](../../../../roots/StructuredBranch.md).
