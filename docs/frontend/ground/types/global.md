# [Глобальные типы (global.d.ts)](../../../../Frontend/Web/ground/types/global.d.ts)

## Назначение
Этот файл содержит расширения глобальных интерфейсов браузера для обеспечения типизации специфических оконных функций проекта (интеграция с AOS, глобальные хендлеры навигации).

---

## Объявленные интерфейсы
Расширение объекта `Window`:
*   **`AOS`**: Доступ к библиотеке Animate On Scroll.
*   **`goTo`**: Глобальная функция навигации Shell.
*   **`toggleSidebar`**: Управление боковым меню.
*   **`handleImageError`**: Обработчик для автоматической замены битых иконок на плейсхолдеры.
