# [API Карточки предмета ([id].ts)](../../../../../Frontend/Web/functions/api/item/[id].ts)

## Назначение
Cloudflare Pages Function для предоставления динамической SEO-страницы предмета через API.

## Функционал
*   **Динамическая генерация**: Создает полный HTML-документ с внедренными мета-данными для конкретного `id` предмета.
*   **Gacha-совместимость**: Позволяет внешним сервисам получать отрендеренную карточку предмета для предпросмотра.
