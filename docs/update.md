# [Скрипт очистки рабочей копии (update.ps1)](../update.ps1)

## Назначение
PowerShell-скрипт (Windows): удаляет всё содержимое каталога, КРОМЕ `.env`, `.github`, `.git`, `update.ps1` — для «чистого» переразвёртывания/обновления рабочей копии.

## Подробное описание
*   Перебирает элементы, берёт во владение (`takeown`), выдаёт права (`icacls`) и рекурсивно удаляет (`Remove-Item -Recurse -Force`).

## AI-контекст
*   ⚠️ Деструктивный служебный скрипт автора под Windows. Сохраняет git/CI/секреты, удаляет остальное (предполагается последующий `git pull`/clone).
