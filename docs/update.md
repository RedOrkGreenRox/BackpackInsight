# [Скрипт очистки рабочей копии (update.ps1)](../update.ps1)

## Назначение
`update.ps1` — это деструктивный служебный скрипт на языке **PowerShell**, предназначенный для использования в среде Windows. Его основная задача — радикальная очистка рабочей директории проекта перед обновлением или переразвертыванием.

---

## Логика работы

Скрипт выполняет следующие действия:
1.  **Фильтрация**: Получает список всех файлов и папок в текущей директории, исключая критически важные для проекта элементы:
    *   `.env` — файл с секретами и настройками.
    *   `.github` — пайплайны CI/CD.
    *   `.git` — история версий проекта.
    *   `update.ps1` — сам скрипт.
2.  **Захват прав (`takeown` / `icacls`)**: Рекурсивно перехватывает права владения и выдает полные права текущему пользователю на все остальные файлы. Это необходимо для удаления файлов, которые могут быть заблокированы процессами (например, Docker или IDE).
3.  **Удаление (`Remove-Item`)**: Рекурсивно и принудительно удаляет все отфильтрованные файлы (включая `Backend`, `Frontend`, `node_modules` и т.д.).

---

## AI-контекст
*   **Осторожно!** Это «ядерный» вариант обновления. После его запуска в папке не останется ничего, кроме Git-истории и секретов. Предполагается, что следующим шагом будет выполнение `git checkout .` или `git pull`.
*   Скрипт специфичен для Windows и требует прав администратора для корректной работы команд `takeown`.
*   В Linux/Unix средах этот файл бесполезен и может быть проигнорирован.
