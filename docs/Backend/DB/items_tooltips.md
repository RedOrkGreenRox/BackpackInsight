# [Тултипы предметов (items_tooltips.json)](../../../Backend/DB/items_tooltips.json)

## Назначение
Файл `items_tooltips.json` — это специализированный справочник текстовых описаний и эффектов для всех предметов игры Backpack Brawl.

## Роль в проекте
Этот файл используется как основной источник данных для **парсеров иконок**:
*   **Бэкенд**: [icon_parser.py](../PlayerData/builds/icon_parser.md) использует эти тексты для извлечения ключевых слов.
*   **Фронтенд**: [icon-parser.ts](../../Frontend/ground/utils/icon-parser.md) преобразует текстовые маркеры (например, `[Poison]`, `[Damage]`) в графические иконки при отрисовке интерфейса.

## Структура
Объект, где ключом является `item_id`, а значением — массив строк или объектов с описанием уровней и эффектов предмета.

## AI-контекст
*   В отличие от основных файлов предметов, этот файл содержит преимущественно текстовую информацию для отображения пользователю.
*   При добавлении новых механик в игру (например, нового типа баффа), соответствующие термины должны быть добавлены сюда, чтобы парсер смог распознать их и заменить на иконки.
