# [История версий БД (Migrations)](../../../../Backend/DB/migrations)

## Назначение
Директория содержит скрипты [**Alembic**](https://alembic.sqlalchemy.org/), которые управляют эволюцией схемы базы данных проекта.

---

## Содержимое
*   [**env.py**](env.md): Скрипт конфигурации окружения миграций.
*   [**versions/**](versions/index.md): Список всех атомарных изменений схемы.
