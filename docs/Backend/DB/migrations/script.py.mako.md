# [Шаблон ревизий (script.py.mako)](../../../../Backend/DB/migrations/script.py.mako)

## Назначение
Этот файл является шаблоном (Mako template), который Alembic использует для генерации новых файлов миграций в папке `versions/`.

## Содержимое
Шаблон определяет структуру будущего Python-файла миграции:
*   Импорт необходимых библиотек (`alembic`, `sqlalchemy`, `sqlmodel`).
*   Метаданные ревизии (`revision`, `down_revision`, `branch_labels`).
*   Пустые функции `upgrade()` и `downgrade()`, которые будут заполнены кодом при запуске `--autogenerate`.

## Настройка
В проект внесены правки в шаблон для поддержки специфики **SQLModel** (например, импорт `sqlmodel` по умолчанию), чтобы генерируемые файлы были совместимы с типизацией проекта.

## AI-контекст
*   Обычно этот файл не требует частого редактирования. Его стоит изменять только если нужно добавить глобальные хуки для всех новых миграций (например, автоматическое добавление комментариев к таблицам).
