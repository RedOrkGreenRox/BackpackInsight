# [Логика игроков и предметов (Backend/PlayerData)](../../../Backend/PlayerData)

## Назначение
Ядро бэкенда. Содержит описание моделей данных игроков, героев и предметов, а также сервисы для их парсинга из игровых логов.

---

## Содержимое
*   [**api.py**](api.md): FastAPI эндпоинты.
*   [**models/**](models/index.md): SQLModel описания сущностей.
*   [**services/**](services/index.md): Бизнес-логика (фабрика профилей).
*   [**data.py**](data.md): Расчеты уровней и справочники.
