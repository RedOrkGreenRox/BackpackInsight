# [Константы (constants.py)](../../../Backend/PlayerData/constants.py)

## Назначение
Файл `constants.py` содержит маппинги технических ключей из JSON-логов игры Backpack Brawl в именованные константы. Это позволяет избежать использования "магических строк" в коде парсеров.

---

## Группы констант

### 1. `TECHNICAL_KEYS`
Словарь ключей из секции `Data`. Содержит сокращения, используемые игровым движком (Unity):
*   `AV` -> App Version
*   `UID` -> User ID
*   `Mem` -> Memory
*   `OS` -> Operating System

### 2. `GAME_KEYS`
Ключи для основной игровой информации:
*   `UL` -> Unlock List (скины, баннеры)
*   `PH` -> Purchase History
*   `Hero` / `Item` — словари сущностей.

### 3. `DEFAULT_VALUES`
Начальные значения для полей (уровень 1, редкость Common), используемые при отсутствии данных в логе.

---

## Связи (Dependencies)
*   **Используется в**: [ProfileFactory](services/ProfileFactory.md) для корректного маппинга JSON в атрибуты модели [Profile](models/Profile.md).

## AI-контекст
*   Если разработчики игры изменят ключи в логе, правки нужно вносить именно в этот файл. Это изолирует логику парсинга от изменений формата данных.
