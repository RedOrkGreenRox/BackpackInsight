# [Генератор Sitemap (sitemap.ts)](../../../../Frontend/Web/functions/api/sitemap.ts)

## Назначение
Динамическая Cloudflare Function для автоматической генерации файла `sitemap.xml`. Она гарантирует, что поисковые роботы (Google, Yandex) всегда знают о существовании всех страниц предметов.

---

## Процесс генерации

1.  **Статические пути**: Добавляет главную страницу, каталог предметов и страницу профиля с высоким приоритетом (0.8–1.0).
2.  **Динамические пути**:
    *   Запрашивает актуальный список предметов через внутренний API.
    *   Генерирует уникальные URL для каждого предмета, используя [SlugService](../../ground/utils/SlugService.md).
3.  **Форматирование**: Собирает данные в стандартный XML-формат, понятный поисковикам.

## Настройки индексации
*   `changefreq`: задает частоту обновления (ежедневно для главной, ежемесячно для предметов).
*   `priority`: расставляет веса страниц для более эффективного краулинга.

---

## AI-контекст
*   Функция кэшируется на Edge-узлах на 1 час (`s-maxage=3600`), что предотвращает лишние запросы к API при каждом визите бота.
*   Использует метод `normalize('NFKD')` для корректной обработки диакритических знаков в названиях предметов при создании слагов.
