# [Оптимизация изображений (optimize-images.js)](../../../Frontend/Web/scripts/optimize-images.js)

## Назначение
Скрипт для автоматизированной обработки и сжатия графических ресурсов проекта. Он обеспечивает конвертацию исходных изображений в современные форматы (AVIF/WebP) для ускорения загрузки сайта.

## Роль в CI/CD
Скрипт запускается перед сборкой проекта, чтобы гарантировать минимальный размер бандла статики.
