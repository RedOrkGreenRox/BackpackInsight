# [Анализ текстов предметов (analyze-item-text.js)](../../../Frontend/Web/scripts/analyze-item-text.js)

## Назначение
Скрипт `analyze-item-text.js` предназначен для глубокого лингвистического анализа базы знаний предметов Backpack Brawl. Он используется для оптимизации поисковых алгоритмов и подготовки данных для семантического поиска.

---

## Возможности

### 1. Токенизация и Нормализация
Скрипт извлекает все текстовые части из JSON-справочника предметов (включая названия, описания и даже ключи объектов) и приводит их к стандартному виду:
*   Разделение CamelCase (например, `criticalChance` -> `critical chance`).
*   Удаление пунктуации и приведение к нижнему регистру.
*   Удаление диакритических знаков (акцентов).

### 2. Фильтрация "Стоп-слов"
Скрипт содержит обширный список `MANUAL_STOPWORDS` на английском и русском языках. Это слова, которые слишком часто встречаются и не несут смысловой нагрузки для поиска (например, `the`, `and`, `when`, `items`, `level`).

### 3. Генерация отчета
Результатом работы является файл `tmp/item-text-statistics.json`, содержащий:
*   Топ-250 самых частотных символов.
*   Топ-250 "сырых" слов.
*   Топ-250 отфильтрованных (значимых) слов.

## AI-контекст
*   Этот скрипт помогает разработчикам понять, какие ключевые слова наиболее важны для системы фильтрации.
*   Данные из отчета могут быть использованы для настройки весов в поисковом движке [fuse.js](https://www.fusejs.io/).
