# [Локализация: английский (en.json)](../../../../Frontend/Web/static/lang/en.json)

## Назначение
Словарь переводов интерфейса на английском: ключ → строка. Зеркало русского словаря по набору ключей.

## Связи (Dependencies)
*   Читается [i18n](../../ground/localization/i18n.md); парный к [ru.json](ru.md).

## AI-контекст
*   Набор ключей обязан совпадать с [ru.json](ru.md). Рассинхрон ключей = отсутствующие переводы в UI.
