# 🖼 Постоянные ассеты (images/const/)

## Назначение
Неизменные общие изображения: `logo`, `menu`, `placeholder` (в AVIF/WebP).

## Связи (Dependencies)
*   `logo`/`menu` прекэшируются [Service Worker](../../sw.md) и используются в [index.html](../../../index.html.md).

## AI-контекст
*   Это «unhashed» стабильные ресурсы — поэтому их безопасно прекэшировать в SW (в отличие от хешированных ассетов сборки).
