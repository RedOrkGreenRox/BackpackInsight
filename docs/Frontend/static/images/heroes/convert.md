# [Конвертация иконок героев (convert.ps1)](../../../../../Frontend/Web/static/images/heroes/convert.ps1)

## Назначение
PowerShell-утилита (Windows, требует `nconvert`): нормализует имена файлов/папок в нижний регистр, удаляет дубликаты скинов (`02-skinXX`) и конвертирует `01-skinXX` исходники в WebP+AVIF, раскладывая по папкам `webp/`/`avif/`.

## Подробное описание (этапы)
1.  Переименование файлов в lowercase (через временное имя, чтобы обойти регистронезависимость FS).
2.  Переименование папок в lowercase.
3.  Удаление изображений вида `*02-skinXX`.
4.  Конвертация `*01-skinXX` → `.webp` и `.avif` (через `nconvert`), удаление исходника только при успехе обоих.

## Связи (Dependencies)
*   Готовит ассеты для форматной стратегии [ImageFormatService](../../../ground/utils/ImageFormatService.md).

## AI-контекст
*   ⚠️ Деструктивный обслуживающий скрипт (удаляет исходники). Имена приводятся к слаг-совместимому виду (см. [SlugService](../../../ground/utils/SlugService.md)). Разовый инструмент подготовки ассетов.
