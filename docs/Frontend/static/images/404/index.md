# 🖼 Фоны страницы 404 (images/404/)

## Назначение
5 фоновых изображений (`00`…`04`) для страницы ошибки — по «редкости», выбираемой случайно при заходе на 404.

## Структура
*   `avif/0N.avif`, `webp/0N.webp` (N = 0…4).

## Связи (Dependencies)
*   Случайный выбор редкости и установка фона: [BackgroundManager](../../../ground/branches/404/_404/background/background.md) → [Shell](../../../ground/roots/Shell.md).

## AI-контекст
*   Индексы `00`…`04` соответствуют вероятностям в `set404Background()` (80%/15%/4%/0.9%/0.1%). Менять количество фонов — синхронно с логикой выбора.
