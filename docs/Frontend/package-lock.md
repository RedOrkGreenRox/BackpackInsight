# [Лок-файл зависимостей (package-lock.json)](../../Frontend/Web/package-lock.json)

## Назначение
Этот файл автоматически генерируется npm и служит для строгой фиксации версий всех сторонних библиотек и их зависимостей.

## Роль в проекте
*   **Воспроизводимость**: Гарантирует, что при развертывании проекта через [Dockerfile](../Dockerfile.md) будут установлены ровно те же версии пакетов, что и при локальной разработке.
*   **Безопасность**: Содержит хеши (integrity) всех пакетов для защиты от подмены кода в репозиториях npm.
