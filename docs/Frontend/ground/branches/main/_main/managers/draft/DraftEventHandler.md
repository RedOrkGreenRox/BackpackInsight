# [События черновиков (DraftEventHandler.ts)](../../../../../../../../Frontend/Web/ground/branches/main/_main/managers/draft/DraftEventHandler.ts)

## Назначение
Менеджер событий для системы черновиков. Отвечает за отслеживание изменений в текстовом поле и своевременную передачу данных на сохранение.

## Функционал
*   **Инициализация**: Находит элемент `#jsonInput` и подписывается на событие `input`.
*   **Синхронизация**: При каждом нажатии клавиши или вставке текста вызывет колбэк `onSave`.
*   **Очистка**: Метод `destroy` гарантирует снятие слушателя, предотвращая дублирование сохранений при повторных визитах на страницу.
