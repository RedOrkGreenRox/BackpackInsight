# [Стиль кнопки Glass UI (button-base.scss)](../../../../../../../../Frontend/Web/ground/branches/main/_main/upload-zone/button-styles/button-base.scss)

## Назначение
Реализует эффект «стеклянного интерфейса» для главной кнопки загрузки.

## Характеристики
*   **Фон**: `rgba(0, 0, 0, 0.5)`.
*   **Рамка**: Тонкая полупрозрачная линия `2px`.
*   **Форма**: Формат «пилюли» (`50px` скругление).
*   **Тень**: Внешняя тень для создания объема над поверхностью страницы.
