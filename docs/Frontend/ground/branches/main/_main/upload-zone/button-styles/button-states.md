# [Состояния кнопки (button-states.scss)](../../../../../../../../Frontend/Web/ground/branches/main/_main/upload-zone/button-styles/button-states.scss)

## Назначение
Описание интерактивных состояний кнопки загрузки.

## Характеристики
*   **Hover**: Активация неонового свечения `--azure` и подъем кнопки на 2px.
*   **Active**: Эффект «вдавливания» (`translateY(2px)`) и затемнение фона.
