# [Drag-and-Drop (DragDropHandler.ts)](../../../../../../../../Frontend/Web/ground/branches/main/_main/upload-zone/handlers/DragDropHandler.ts)

## Назначение
Реализация интерфейса перетаскивания файлов (Drag-and-Drop) для зоны загрузки профиля.

---

## Функционал

### 1. Визуальная обратная связь
При наведении файла на зону загрузки к элементу добавляется CSS-класс `.drag-over`. Это активирует стили свечения и изменения фона, подсказывая пользователю, что файл можно отпускать.

### 2. Обработка событий
Скрипт подавляет стандартное поведение браузера (открытие файла в новой вкладке) для событий `dragenter`, `dragover`, `dragleave` и `drop`.

### 3. Загрузка файла
При срабатывании события `drop` из объекта `DataTransfer` извлекается первый файл и передается в колбэк `onFileRead`.

---

## AI-контекст
*   Класс обеспечивает "бесшовный" пользовательский опыт, позволяя игрокам просто перетягивать логи из папки с игрой прямо в браузер.
*   Все события очищаются в методе `destroy()` при уходе с главной страницы.
