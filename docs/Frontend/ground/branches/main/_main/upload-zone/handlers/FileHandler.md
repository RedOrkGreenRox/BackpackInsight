# [Обработчик файлов (FileHandler.ts)](../../../../../../../../Frontend/Web/ground/branches/main/_main/upload-zone/handlers/FileHandler.ts)

## Назначение
Статический сервис для низкоуровневой работы с объектами `File`. Он отвечает за чтение контента и базовую проверку расширений.

---

## Функционал

### 1. Чтение (`readAndProcessFile`)
Использует современный API `file.text()`, который возвращает промис с текстовым содержимым файла. Это более чистая и быстрая альтернатива старому `FileReader`.

### 2. Валидация типа
Метод `isValidJsonFile` проверяет файл по двум критериям:
1.  MIME-тип `application/json`.
2.  Расширение имени файла `.json`.

---

## AI-контекст
*   Сервис является "чистым" — он не имеет состояния и не зависит от DOM, что делает его легко тестируемым.
*   Используется совместно с [ClipboardHandler](ClipboardHandler.md) и [DragDropHandler](DragDropHandler.md).
