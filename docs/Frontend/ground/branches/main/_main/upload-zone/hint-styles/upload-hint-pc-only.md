## Назначение
Управление видимостью подсказок, актуальных только для управления мышью.

## Характеристики
*   **Действие**: Скрывает блок "Перетащите экспорт сюда" на экранах уже `1024px`, так как на мобильных устройствах механика Drag-and-Drop для файлов в браузере не является стандартным сценарием.
