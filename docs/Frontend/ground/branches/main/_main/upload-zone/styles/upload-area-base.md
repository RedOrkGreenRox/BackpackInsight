# [База зоны загрузки (upload-area-base.scss)](../../../../../../../../Frontend/Web/ground/branches/main/_main/upload-zone/styles/upload-area-base.scss)

## Назначение
Определяет базовую геометрию и оформление рамки зоны перетаскивания.

## Характеристики
*   **Стиль**: Пунктирная рамка (`dashed`) белого цвета с прозрачностью 0.4.
*   **Высота**: Минимальная высота 250px для удобства перетаскивания.
*   **Эффекты**: Внутренняя тень (`inset box-shadow`) для создания эффекта углубления.
