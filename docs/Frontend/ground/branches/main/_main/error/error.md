# [Стили ошибок главной (error.scss)](../../../../../../../Frontend/Web/ground/branches/main/_main/error/error.scss)

## Назначение
Детальные стили для блока уведомлений об ошибках на главной странице.

---

## Ключевые элементы
*   **`.error`**: Базовый контейнер с красным фоном.
*   **`.validation-error-header`**: Горизонтальный flex-блок для заголовка и кнопки закрытия.
*   **`.error-dismiss-btn`**: Прозрачная кнопка-крестик для ручного скрытия уведомления.
*   **`.validation-error-footer`**: Блок для вывода технической информации об ошибке синтаксиса JSON.
