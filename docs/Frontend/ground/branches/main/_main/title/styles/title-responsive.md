## Назначение
Масштабирует заголовок в зависимости от размера экрана.

## Характеристики
*   **Планшеты**: `2.2em`.
*   **Телефоны**: `1.8em` с уменьшенным межсимвольным интервалом для предотвращения разрывов слов.
