## Назначение
Специфические анимации для элементов главной страницы.

## Список анимаций
*   **`glow-pulse`**: Плавное изменение тени (`box-shadow`) в акцентном цвете `--azure`. Используется для привлечения внимания к кнопке загрузки или текстовому полю при наведении.
