# [Цвета редкостей в фильтрах (_rarity-colors.scss)](../../../../../../../Frontend/Web/ground/branches/items/_items/chips/_rarity-colors.scss)

## Назначение
Специализированная надстройка над фильтр-чипсами, которая окрашивает их названия в соответствии с игровой редкостью предмета.

## Особенности
*   Применяет переменные типа `--rarity-common-text` к тексту чипсов внутри блока `#filterRarities`.
*   При активации чипса текст становится белым для максимального контраста на голубом фоне.
