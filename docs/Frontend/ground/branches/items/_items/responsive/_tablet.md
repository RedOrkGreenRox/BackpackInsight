# [Планшетная адаптация Вики (_tablet.scss)](../../../../../../../Frontend/Web/ground/branches/items/_items/responsive/_tablet.scss)

## Назначение
Коррекция элементов библиотеки предметов для экранов шириной до 768px.

---

## Изменения
*   **Заголовок**: Использование `clamp()` для динамического размера главного названия.
*   **Панель фильтров**: Добавление горизонтальных отступов (`10px`), чтобы контент не прилипал к краям экрана.
*   **Инпут**: Масштабирование шрифта до `1rem`.
