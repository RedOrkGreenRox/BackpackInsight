# [Группа фильтров (_filter-group.scss)](../../../../../../../Frontend/Web/ground/branches/items/_items/filters/_filter-group.scss)

## Назначение
Стилизация логического блока внутри фильтра (заголовок + список чипсов).

---

## Элементы
*   **`.filter-label`**: Небольшой жирный заголовок над группой выбора.
*   **`.filter-multiselect`**: Контейнер с `flex-wrap: wrap`, обеспечивающий автоматический перенос чипсов на новую строку.
