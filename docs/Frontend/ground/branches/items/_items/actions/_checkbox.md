# [Стили чекбоксов (_checkbox.scss)](../../../../../../../Frontend/Web/ground/branches/items/_items/actions/_checkbox.scss)

## Назначение
Стилизация элементов типа "флажок" для фильтрации по бинарным параметрам (например, "Только покупаемые").

## Характеристики
*   **Дизайн**: Использует `accent-color: var(--azure)` для приведения стандартного браузерного чекбокса к общему стилю проекта.
*   **Типографика**: Текст метки имеет мягкую тень для читаемости.
