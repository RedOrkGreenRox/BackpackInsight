# [Контейнер 404: стили (container.scss)](../../../../../../../Frontend/Web/ground/branches/404/_404/container/container.scss)

## Назначение
Стилизация центрального информационного блока.

## Характеристики
*   **Центрирование**: Идеальное позиционирование по центру экрана через `absolute` и `translate(-50%, -50%)`.
*   **Адаптивность**: При малой высоте экрана (`max-height: 600px`) переключается в режим `relative`, чтобы контент не обрезался панелями навигации и был доступен через внутренний скролл.
