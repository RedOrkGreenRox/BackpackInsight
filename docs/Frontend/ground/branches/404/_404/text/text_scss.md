# [Текст 404: стили (text.scss)](../../../../../../../Frontend/Web/ground/branches/404/_404/text/text.scss)

## Назначение
Стилизация текстового сообщения на странице ошибки.

## Характеристики
*   **Читаемость**: Использует жесткую четырехстороннюю тень (`text-shadow`), создающую черный контур вокруг букв. Это гарантирует 100% читаемость даже на самом ярком фоне (Legendary).
*   **Типографика**: Шрифт `Signika`, вес 500, адаптивный размер через `clamp`.
