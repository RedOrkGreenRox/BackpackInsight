# [Фон 404: логика (background.ts)](../../../../../../../Frontend/Web/ground/branches/404/_404/background/background.ts)

## Назначение
Класс `BackgroundManager` управляет системой «редкостных» фонов для страницы 404, имитируя механику выпадения предметов в игре Backpack Brawl.

---

## Ключевая логика

### 1. Вероятности (Gacha Logic)
При каждом входе на страницу генерируется случайное число, определяющее фон:
*   **04 (Legendary)**: 0.1% шанс.
*   **03 (Epic)**: 0.9% шанс.
*   **02 (Rare)**: 4% шанс.
*   **01 (Uncommon)**: 15% шанс.
*   **00 (Common)**: 80% шанс.

### 2. Управление Оболочкой
Менеджер вызывает [**Shell.ts**](../../../../roots/Shell.md) для физической замены изображения в DOM.

### 3. Сохранение при смене языка
Реализована логика `preservedRarity`. Если пользователь меняет язык, находясь на странице 404, менеджер блокирует генерацию нового случайного числа, чтобы фон не "мигал" и оставался прежним.
