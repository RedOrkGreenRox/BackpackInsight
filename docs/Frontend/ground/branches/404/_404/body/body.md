# [Основа страницы 404 (body.scss)](../../../../../../../Frontend/Web/ground/branches/404/_404/body/body.scss)

## Назначение
Базовые настройки для тела страницы ошибки.

## Характеристики
*   **Скролл**: Принудительно отключает прокрутку (`overflow: hidden`), превращая страницу в статичную игровую заставку.
*   **Высота**: Использует современную единицу `100dvh` для идеального заполнения экрана мобильных устройств с учетом панелей браузера.
