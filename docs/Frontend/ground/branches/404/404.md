# [Стили 404 (404.scss)](../../../../../Frontend/Web/ground/branches/404/404.scss)

## Назначение
Главный агрегатор стилей для страницы ошибки. Координирует сборку всех визуальных компонентов: от полноэкранного фона до типографики.

## Связи (Dependencies)
Использует `@use` для подключения:
*   [**background/background.scss**](_404/background/background.md)
*   [**overlay/overlay.scss**](_404/overlay/overlay.md)
*   [**body/body.scss**](_404/body/body.md)
*   [**container/container.scss**](_404/container/container.md)
