# [Страница 404 (NotFoundBranch.ts)](../../../../../Frontend/Web/ground/branches/404/NotFoundBranch.ts)

## Назначение
Бранч для обработки несуществующих путей. 

## Особенности
*   **Редкостный фон**: При каждом посещении страницы генерируется случайный фон с разным шансом редкости (аналогично механике игры).
*   **Кнопка возврата**: Содержит интерактивный элемент для перехода на главную.
