# [Статистика игрока (_player-stats.scss)](../../../../../../../Frontend/Web/ground/branches/itemDetail/_itemDetail/player-stats/_player-stats.scss)

## Назначение
Стилизация блока с персональными данными о предмете (уровень игрока, наличие карт).

## Характеристики
*   **Контраст**: Значения (например, "Lvl 15") выделяются более ярким цветом.
*   **Макет**: Вертикальный список параметров с разделением через отступы.
