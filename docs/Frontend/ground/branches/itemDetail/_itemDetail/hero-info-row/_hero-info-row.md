## Назначение
Стилизация компактной строки с иконкой героя, к которому привязан предмет.

## Характеристики
*   **Иконка**: Увеличенный размер `2.8em` для акцентирования связи с персонажем.
*   **Выравнивание**: Использует flexbox для центрирования иконки по вертикали.
