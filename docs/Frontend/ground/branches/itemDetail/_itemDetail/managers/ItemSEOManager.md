# [SEO Менеджер предмета (ItemSEOManager.ts)](../../../../../../../Frontend/Web/ground/branches/itemDetail/_itemDetail/managers/ItemSEOManager.ts)

## Назначение
Отвечает за динамическое обновление мета-данных страницы под конкретный предмет. Это обеспечивает индексацию каждой карточки предмета поисковыми системами.

---

## Функционал

### 1. Мета-теги и OpenGraph
Автоматически заполняет:
*   **Title**: Название предмета + Название проекта.
*   **Description**: Первые 160 символов описания предмета.
*   **OG:Image**: Прямая ссылка на AVIF/WebP иконку предмета для красивых превью в соцсетях.

### 2. Структурированные данные (JSON-LD)
Генерирует объект `Schema.org/Thing`, который содержит:
*   Редкость и стоимость предмета.
*   Типы (категории).
*   Связь с брендом «Backpack Brawl».

---

## Связи (Dependencies)
*   Использует [`MetaService`](../../../../utils/MetaService.md) для манипуляции тегами `<head>`.
