# [Навигация по предметам (ItemNavigationManager.ts)](../../../../../../../Frontend/Web/ground/branches/itemDetail/_itemDetail/managers/ItemNavigationManager.ts)

## Назначение
Реализует возможность переключения между предметами (кнопки «Влево/Вправо») без возврата к общему списку.

---

## Ключевая логика

### 1. Расчет порядка (`calculate`)
Менеджер анализирует кэшированные списки в `sessionStorage`:
*   В **Профиле**: Порядок определяется текущей пользовательской сортировкой.
*   В **Вики**: Порядок определяется результатами последнего поиска и примененными фильтрами.
Благодаря этому кнопки «Вперед/Назад» всегда ведут на логически следующий предмет из того списка, который пользователь видел последним.

### 2. Поддержка контекста профиля
При навигации внутри профиля менеджер автоматически подтягивает данные о прокачке (`playerItem`) следующего предмета, чтобы страница не теряла персонализацию.

---

## AI-контекст
*   Сравнение предметов ведется через [**SlugService**](../../../../utils/SlugService.md), что гарантирует корректную работу ссылок для названий со спецсимволами (напр. `Fool's Gold`).
