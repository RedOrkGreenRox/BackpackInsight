# [Загрузчик данных предмета (ItemDataLoader.ts)](../../../../../../../Frontend/Web/ground/branches/itemDetail/_itemDetail/managers/ItemDataLoader.ts)

## Назначение
Класс для асинхронного получения полной информации о предмете. Он обеспечивает надежную загрузку данных, используя систему кэширования и обработку различных сетевых состояний.

---

## Функционал
*   **Слаг-поиск**: Принимает "слаг" предмета (из URL) и ищет его через [**ItemsCacheService**](../../../../utils/ItemsCacheService.md).
*   **События**: Поддерживает три типа уведомлений:
    1.  `onLoaded`: Данные успешно получены.
    2.  `onNotFound`: Предмет не существует в текущей версии базы знаний.
    3.  `onError`: Произошла сетевая ошибка.
