# [Детализация предмета (ItemDetailBranch.ts)](../../../../../Frontend/Web/ground/branches/itemDetail/ItemDetailBranch.ts)

## Назначение
Страница глубокого анализа конкретного предмета. Отображает рецепты крафта, полные боевые характеристики и дерево зависимостей.

---

## Ключевая логика

### 1. Загрузка данных (`ItemDataLoader`)
Динамически подгружает информацию о предмете по его имени из URL-параметра. Если предмет не найден, автоматически перенаправляет на 404.

### 2. Визуализация крафта
Отрисовывает интерактивные схемы:
*   **Ингредиенты**: Из чего собирается предмет.
*   **Результаты**: Во что может превратиться этот предмет в будущем.

---

## AI-контекст
*   Этот бранч генерирует богатые SEO-метаданные (JSON-LD) для того, чтобы карточки предметов красиво отображались в результатах поиска Google.
