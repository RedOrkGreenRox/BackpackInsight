## Назначение
Описание сетки (Grid) для отображения коллекции персонажей в профиле.

## Конфигурация
*   **Десктоп**: 3 колонки.
*   **Планшеты**: 2 колонки.
*   **Мобильные**: 1 колонка (полная ширина).
*   **Геометрия**: Максимальная ширина ограничена `1200px` для поддержания композиции на широких мониторах.
