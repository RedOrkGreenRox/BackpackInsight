# [Основная карточка героя (_card.scss)](../../../../../../../Frontend/Web/ground/branches/profile/_profile/main-heroes-grid/_card.scss)

## Назначение
Детальная стилизация большой карточки персонажа в основной секции профиля.

---

## Визуальные характеристики
*   **Фон**: Глубокий черный с прозрачностью (`0.6`).
*   **Ховер**: При наведении карточка приподнимается на 5px и активируется комбинированная тень: внешнее размытие (glow) в цвете `--azure` и жесткая падающая тень для объема.
*   **Кнопки скинов**: Абсолютно позиционированные невидимые кнопки слева и справа (`.skin-btn`). Они проявляются легким фоном при наведении, обеспечивая навигацию по обликам героя.

---

## Адаптивность
*   **< 600px**: Карточка переключается в горизонтальный режим (`flex-direction: row`). Это позволяет эффективно использовать высоту экрана на мобильных устройствах.
