# [Заголовки секций (_section-title.scss)](../../../../../../../Frontend/Web/ground/branches/profile/_profile/section-title/_section-title.scss)

## Назначение
Стилизация крупных заголовков разделов ("Персонажи", "Предметы") на странице профиля.

## Характеристики
*   **Размер**: `2rem`.
*   **Цвет**: Использует `--text-default-color` (бежевый).
*   **Тень**: Глубокая тень `0 2px 4px rgba(0, 0, 0, 0.8)` для отделения текста от игровых фонов.
*   **Выравнивание**: По центру.
