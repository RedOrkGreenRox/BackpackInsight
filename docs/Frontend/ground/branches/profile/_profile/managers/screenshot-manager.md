# [Менеджер скриншотов (screenshot-manager.ts)](../../../../../../../Frontend/Web/ground/branches/profile/_profile/managers/screenshot-manager.ts)

## Назначение
Реализует функционал сохранения аналитической карточки игрока в виде изображения (PNG). Это позволяет пользователям делиться своими результатами вне приложения.

---

## Ключевая логика

### 1. Ленивая загрузка (Optimization)
Библиотека `html2canvas` весит достаточно много, поэтому она загружается динамически только при наведении на кнопку "Сохранить" или при клике.

### 2. Смена макета для рендеринга
Поскольку интерфейс адаптивен, прямое снятие скриншота на мобильном устройстве привело бы к получению вертикальной картинки. Менеджер применяет "временные стили": на доли секунды расширяет блок профиля до эталонных 900x600 пикселей, делает снимок и мгновенно возвращает исходный вид.

### 3. Настройки качества
Снимок генерируется с параметром `scale: 2`. Это обеспечивает высокую четкость текста и иконок на Retina-дисплеях и при печати.

---

## AI-контекст
*   Использует `useCORS: true` для корректного захвата иконок предметов, загруженных с внешних доменов или CDN.
*   Метод `destroy` предотвращает утечки памяти, очищая слушатели событий с кнопки.
