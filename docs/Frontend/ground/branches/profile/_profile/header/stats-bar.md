# [Панель ресурсов (stats-bar.ts)](../../../../../../../Frontend/Web/ground/branches/profile/_profile/header/stats-bar.ts)

## Назначение
Рендерер горизонтальной панели с основными игровыми показателями: Уровнем, Опытом, Трофеями, Гемами и Монетами.

---

## Особенности
*   **Локализация чисел**: Использует `toLocaleString()` для добавления разделителей тысяч (например, `10 000` вместо `10000`).
*   **Суммарные трофеи**: Автоматически складывает базовые трофеи и бонусные (`data.trophy + data.bonus_trophy`).

---

## Связи (Dependencies)
*   Использует иконки из директории `/images/profile/`.
