# [Мини-сетка героев (_stats-heroes-grid.scss)](../../../../../../../Frontend/Web/ground/branches/profile/_profile/header/_stats-heroes-grid.scss)

## Назначение
Описание макета для компактной сетки героев, расположенной внутри шапки профиля.

## Характеристики
*   **Дисплей**: `grid`.
*   **Колонки**: Ровно 3 колонки (`minmax(80px, 1fr)`) на десктопе.
*   **Мобильность**: Переключается на 2 колонки для предотвращения чрезмерного сжатия контента.
*   **Эффект**: Тень от всей сетки усиливается при наведении, создавая эффект "парения" над заголовком.
