# [Контейнер заголовка профиля (_header.scss)](../../../../../../../Frontend/Web/ground/branches/profile/_profile/header/_header.scss)

## Назначение
Главный макет (Layout) верхней аналитической карточки игрока. Координирует расположение никнейма, ресурсов и мини-карточек героев.

---

## Ключевые правила
*   **Макет**: Гибкий (`flex-wrap: wrap`) с распределением `space-between`.
*   **Фон**: Поддерживает динамическую подгрузку изображений арен. Имеет внутреннее затемнение (`inset shadow`) для повышения контраста белого текста.
*   **Геометрия**: Скругление углов 25px, максимальная ширина 1100px.

---

## Адаптивность
*   **Планшеты**: Переключается в вертикальный режим (`flex-direction: column`) с выравниванием по центру.
*   **Landscape**: На очень низких экранах уменьшает отступы и размеры шрифтов до минимума для экономии вьюпорта.

---

## Связи (Dependencies)
Импортирует все атомарные стили элементов заголовка:
*   `_nickname.scss`
*   `_stats-player-grid.scss`
*   `_stats-heroes-wrapper.scss`
*   ... и другие.
