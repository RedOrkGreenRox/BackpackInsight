# [Веса редкостей (rarity-weights.ts)](../../../../../../../Frontend/Web/ground/branches/profile/_profile/utils/rarity-weights.ts)

## Назначение
Словарь числовых значений, используемый для математического сравнения редкостей предметов при сортировке.

## Весовые коэффициенты
*   `Unique`: 100 (самый высокий приоритет)
*   `Relic`: 80
*   `Legendary`: 60
*   ...
*   `Special`: 20 (самый низкий приоритет)

---

## AI-контекст
*   Разделение редкостей по весам позволяет реализовать логику «чем ценнее предмет, тем выше он в списке», даже если уровни предметов одинаковы.
