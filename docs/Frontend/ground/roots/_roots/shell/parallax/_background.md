# [Стили фона и Параллакса (_background.scss)](../../../../../../../Frontend/Web/ground/roots/_roots/shell/parallax/_background.scss)

## Назначение
Описание стилей для полноэкранного фона приложения. Обеспечивает поддержку эффекта глубины и корректное поведение при скролле.

---

## Ключевые элементы

### `.background-image`
*   **Слой**: Самый нижний (`z-index: 0`).
*   **Трансформации**: Оптимизировано через `will-change: transform` для плавной работы параллакса.
*   **Изображение**: Картинка внутри растягивается на 110% и центрируется, чтобы при смещении параллаксом не было видно пустых краев.

### `.lock-scroll`
Технический класс, который навешивается на `<body>` при открытии модальных окон или меню. Он блокирует скролл, сохраняя текущую позицию через CSS-переменную `--scroll-y`.

---

## Оптимизация
В режиме [**Low-Res Mode**](../_low-res.md) фильтры размытия на фоне отключаются для экономии ресурсов GPU.
