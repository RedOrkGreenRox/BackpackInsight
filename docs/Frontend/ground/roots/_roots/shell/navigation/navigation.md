# [Навигация Shell (navigation.ts)](../../../../../../../Frontend/Web/ground/roots/_roots/shell/navigation/navigation.ts)

## Назначение
Глобальный контроллер переходов. Содержит вспомогательные функции для навигации в обход SPA-роутера (для полной перезагрузки или жестких редиректов).

---

## Функционал
*   **`goTo(url)`**: Осуществляет переход с анимацией исчезновения (`.leaving`).
*   **Восстановление**: Обрабатывает событие `pageshow`, чтобы вернуть интерфейс в рабочее состояние после использования кнопки "Назад" в браузере.
