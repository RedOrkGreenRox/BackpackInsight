# [Обертка элементов управления (_controls-wrapper.scss)](../../../../../../../Frontend/Web/ground/roots/_roots/shell/navigation/_controls-wrapper.scss)

## Назначение
Контейнер для верхнеуровневых кнопок интерфейса (меню, переключатели). Отвечает за их позиционирование и адаптивное поведение.

---

## Ключевые правила
*   **Позиционирование**: Фиксированное (`fixed`) в правом верхнем углу. Использует `clamp()` для динамических отступов в зависимости от вьюпорта.
*   **Z-index**: `1100` — гарантирует, что кнопки управления всегда находятся поверх основного контента.
*   **Состояние при открытом меню**: Обертка плавно исчезает (`opacity: 0`), чтобы не перекрывать контент сайдбара.

---

## Адаптивность
На мобильных устройствах (`max-width: 1100px`) обертка сбрасывает фоновые эффекты и блюр, превращаясь в чистый контейнер для кнопки-бургера, прижатый к правому краю.
