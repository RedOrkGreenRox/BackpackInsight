# [Переключатель языка (_lang-switcher.scss)](../../../../../../../Frontend/Web/ground/roots/_roots/shell/sidebar/_lang-switcher.scss)

## Назначение
Стилизация кнопки выбора языка (RU/EN), расположенной в нижней части сайдбара.

---

## Особенности реализации
*   **Позиционирование**: Абсолютное внизу сайдбара, центрированное по горизонтали.
*   **Дизайн**: Скругленные углы (20px), полупрозрачный фон с эффектом размытия (`blur(10px)`).
*   **Адаптив**: Использует `env(safe-area-inset-bottom)` для корректного отображения на безрамочных смартфонах (iPhone), предотвращая наложение на системную полоску навигации.

## Анимации
При открытии меню кнопка плавно выплывает снизу (`slideUpFadeIn`) с небольшой задержкой (0.3с).
