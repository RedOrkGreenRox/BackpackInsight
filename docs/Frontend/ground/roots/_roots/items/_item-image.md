# [Иконка предмета (_item-image.scss)](../../../../../../Frontend/Web/ground/roots/_roots/items/_item-image.scss)

## Назначение
Стили для контейнера и самого изображения (иконки) предмета. Обеспечивает правильное масштабирование и визуальные эффекты.

---

## Описание элементов

### `.item-image-wrapper`
*   **Геометрия**: Квадратный контейнер с использованием `aspect-ratio: 1`.
*   **Центрирование**: Использует flexbox для идеального выравнивания иконки по центру.

### `.item-icon`
*   **Эффекты**: Добавлена тень `drop-shadow` для отделения плоской иконки от фона карточки.
*   **Интерактивность**: При наведении на карточку (`:hover`) иконка плавно увеличивается на 10% (`scale(1.1)`).

### `.no-image` (Плейсхолдер)
Если иконка отсутствует, контейнер переходит в режим плейсхолдера: иконка становится черно-белой и полупрозрачной.

---

## Связи (Dependencies)
