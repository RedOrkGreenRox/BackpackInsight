# [Переменные дизайна (_vars.scss)](../../../../../Frontend/Web/ground/roots/_roots/_vars.scss)

## Назначение
Глобальный конфигурационный файл CSS-переменных (CSS Custom Properties), определяющий цветовую схему, эффекты свечения и базовые метрики типографики для всего приложения.

---

## Список переменных

### Цвета и Акценты
*   **`--azure-raw`**: `0, 212, 255` — RGB компоненты основного цвета (используется для динамического изменения прозрачности через `rgba()`).
*   **`--azure`**: Основной голубой цвет интерфейса.
*   **`--azure-glow-low`**: Мягкое внешнее и внутреннее свечение для интерактивных элементов.
*   **`--azure-glow-high`**: Интенсивное свечение для активных состояний или важных акцентов.
*   **`--border`**: Стандартная полупрозрачная рамка (1px white 20%).

### Типографика Описаний
Эти переменные используются преимущественно в тултипах предметов:
*   **`--text-header-color`**: `#FFA500` (оранжевый) — цвет заголовков и триггеров.
*   **`--text-default-color`**: `#F5F5DC` (бежевый) — цвет основного пояснительного текста.
*   **`--wiki-stats-text-size`**: `1rem` — базовый размер шрифта для блоков характеристик.

---

## Связи (Dependencies)
Импортируется в:
*   [`_roots.scss`](../_roots.md) — для распространения переменных на всё приложение.
*   Многие локальные компоненты (через наследование `:root`).

## AI-контекст
*   Переменная `--azure-raw` позволяет создавать эффекты типа `background: rgba(var(--azure-raw), 0.5)`, что невозможно при использовании HEX-кодов.
*   Изменение одной переменной здесь мгновенно обновляет всю цветовую гамму приложения.
