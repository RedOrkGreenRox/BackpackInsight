# [Сайдбар и Навигация (sidebar.ts / navigation.ts)](../../../../../Frontend/Web/ground/roots/_roots/shell/sidebar/sidebar.ts)

## Назначение
Эта группа файлов управляет основным навигационным меню приложения.

## Функционал
*   **`sidebar.ts`**: Менеджер боковой панели. Отвечает за открытие/закрытие меню на мобильных устройствах.
*   **`navigation.ts`**: Логика переключения активных вкладок и подсветка текущего раздела.
*   **`_sidebar.scss`**: Стилизация панели с использованием полупрозрачных фонов (glassmorphism).
