# [Кэширование профиля (profileCacheUtils.ts)](../../../../Frontend/Web/ground/roots/profileCacheUtils.ts)

## Назначение
Набор утилит для управления жизненным циклом данных игрока в хранилищах браузера (`sessionStorage` и `localStorage`). Обеспечивает сохранность данных профиля при навигации и обновлении страницы.

---

## Ключевая логика

### 1. Очистка кэша
*   **`clearAllProfileCache()`**: Удаляет все данные профиля (инвентарь, статы, состояние) из обоих типов хранилищ.
*   **`clearCacheOnNavigation()`**: Интеллектуальный метод, который решает, нужно ли чистить кэш при переходе. Он сохраняет данные при возврате кнопкой "Назад" или переходе внутри сайта, но сбрасывает их, если пользователь начинает новую сессию.

### 2. Проверка состояния
*   **`hasProfileCache()`**: Проверяет наличие любых признаков загруженного профиля, что используется роутером для принятия решения об автоматическом редиректе.

---

## AI-контекст
*   Использование `sessionStorage` гарантирует, что данные игрока будут удалены после закрытия вкладки браузера (конфиденциальность).
*   Логика `clearCacheOnNavigation` предотвращает потерю данных при переходе между деталями предметов и основной страницей профиля.
