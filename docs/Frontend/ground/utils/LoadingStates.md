# [Состояния загрузки (LoadingStates.ts)](../../../../Frontend/Web/ground/utils/LoadingStates.ts)

## Назначение
Сервис для создания визуальных индикаторов ожидания. Он реализует концепцию **Skeleton Screens** (скелетные экраны), которые делают процесс загрузки данных психологически быстрее для пользователя.

---

## Возможности

### 1. Skeleton-карточки
Метод `createCardSkeleton` генерирует анимированные серые блоки, имитирующие структуру карточек предметов. Это используется в библиотеке и профиле до получения ответа от сервера.

### 2. Скелет профиля
Специальная заготовка для верхней части страницы игрока, включающая заглушки под аватар, ресурсы и сетку персонажей.

### 3. Динамические индикаторы
*   **Спиннеры**: Вращающиеся круги трех размеров для кнопок и полноэкранных переходов.
*   **Прогресс-бары**: Используются для визуализации процесса анализа тяжелых логов.

---

## AI-контекст
*   Все скелетные экраны поддерживают библиотеку анимаций **AOS**, плавно появляясь при скролле.
*   Интегрирован с [**i18n**](../localization/i18n.md) для корректного отображения текста "Загрузка..." на разных языках.
