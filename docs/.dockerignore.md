# [Исключения сборки Docker (.dockerignore)](../.dockerignore)

## Назначение
Исключает из контекста сборки Docker ненужные/тяжёлые каталоги, ускоряя билд и уменьшая образ.

## Содержимое
*   `**/node_modules`, `**/dist`, `**/__pycache__`, `.git`, `Frontend/Web/dist`, `Frontend/Web/node_modules`.

## AI-контекст
*   Зависимости ставятся внутри образов ([Dockerfile](backend/playerdata/Dockerfile.md)/[Dockerfile](frontend/Dockerfile.md)), поэтому локальные `node_modules`/`dist` в контекст не нужны.
