# [Генератор карты (generate_structure.py)](../../scripts/generate_structure.py)

## Назначение
Скрипт `generate_structure.py` автоматически создает и обновляет файл [docs/structure.md](../structure.md). Он служит для визуализации иерархии проекта и навигации по документации.

---

## Возможности

1.  **Индикация состояния**: 
    *   `📄` — файл задокументирован.
    *   `❌` — документация отсутствует (`MISSING DOC`).
2.  **Человекочитаемые имена**: Скрипт использует словарь `BEAUTIFUL_NAMES` для замены технических имен файлов и папок на понятные заголовки (например, `PlayerData` -> `Логика игроков и предметов`).
3.  **Умная фильтрация**: Скрывает служебные папки (`.git`, `node_modules`, `__pycache__`) и ассеты (картинки, шрифты), если они не имеют собственного файла описания.
4.  **Кликабельные папки**: Если у папки есть вводный файл `index.md`, название папки в дереве становится ссылкой.

---

## Исключения
*   `ARENA.MD` — meta-файл правил для агентов, не является исходным кодом и не включается в `docs/structure.md`.

## AI-контекст
*   Скрипт автоматически обрабатывает переименование файлов. После любого перемещения файлов в проекте запустите этот скрипт, чтобы обновить карту ссылок.
*   Поддерживает поиск документации даже при различии в регистре имен (например, `api.py` -> `api.md`).
