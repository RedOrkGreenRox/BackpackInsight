# [Проверка индексов БД (verify_indexes.py)](../../scripts/verify_indexes.py)

## Назначение
Диагностический скрипт: подключается к PostgreSQL и проверяет наличие нужных индексов (в т.ч. GIN/`pg_trgm` для поиска).

## Подробное описание
*   Вручную парсит `.env` в `os.environ`.
*   Формирует `DATABASE_URL` из `POSTGRES_*` (дефолты совпадают с docker-compose).
*   Подключается через SQLAlchemy и проверяет/выводит индексы.

## Связи (Dependencies)
*   Индексы создаёт [api.py (create_indexes)](../backend/playerdata/api.md) и миграция Alembic `0002`.

## AI-контекст
*   `POSTGRES_SERVER` по умолчанию `localhost` — скрипт рассчитан на запуск с хоста (не из контейнера).
