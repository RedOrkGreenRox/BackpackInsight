# [Авто-коммит и пуш (git_push.py)](../../scripts/git_push.py)

## Назначение
Служебный скрипт автоматизации git: формирует коммит с временной меткой и пушит. Объясняет происхождение коммитов вида «Automated push: <дата>» в истории.

## Подробное описание
*   Принудительно ставит UTF-8 на stdout (фикс cp1251 на Windows).
*   `run_git(args, allow_fail)` — обёртка над `git` с обработкой ошибок.
*   Проверяет наличие изменений; формирует сообщение коммита; коммитит и синхронизируется с remote (`allow_fail` на pull/push при отсутствии ветки/конфликтах).

## AI-контекст
*   Dev-утилита автора. Из-за неё история git состоит из однотипных автокоммитов — учитывайте при аудите изменений.
