# [Пуш изменений (git_push.py)](../../scripts/git_push.py)

## Назначение
Служебный скрипт для автоматизации процесса отправки изменений в Git-репозиторий.

## Функционал
1.  **Синхронизация структуры**: Перед пушем автоматически запускает [generate_structure.py](generate_structure.md), гарантируя, что файл `structure.md` всегда актуален.
2.  **Git-цепочка**: Выполняет последовательность команд `git add .`, `git commit -m "..."` и `git push`.
