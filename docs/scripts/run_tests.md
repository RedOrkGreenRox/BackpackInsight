# [Автоматический запуск тестов (run_tests.py)](../../scripts/run_tests.py)

## Назначение
Служебный скрипт-обертка для запуска всего набора тестов проекта с правильной настройкой окружения.

## Функционал
1.  **Настройка путей**: Автоматически добавляет корень проекта в `PYTHONPATH`, чтобы тесты могли импортировать модули из папки `Backend`.
2.  **Запуск pytest**: Инициирует выполнение тестов на основе настроек в [pytest.ini](../../pytest.md).
3.  **Изоляция**: Гарантирует, что тесты запускаются с корректными переменными окружения (например, принудительное использование SQLite).
