# [Скриптовый конфиг тестов (pytest.ini)](../../scripts/pytest.ini)

## Назначение
Локальный файл конфигурации `pytest` для папки служебных скриптов.

## Настройки
*   **`testpaths = tests`**: Указывает на папку с тестами для самих скриптов автоматизации.
*   **`addopts = -v`**: Включает подробный (verbose) вывод хода тестирования.
