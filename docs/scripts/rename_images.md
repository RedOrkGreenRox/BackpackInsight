# [Переименование изображений (rename_images.py)](../../scripts/rename_images.py)

## Назначение
Скрипт для массового переименования и нормализации файлов изображений в папке статики.

## Функционал
1.  **Очистка имен**: Приводит имена файлов к нижнему регистру, заменяет пробелы на дефисы и удаляет спецсимволы.
2.  **SEO-оптимизация**: Гарантирует, что URL изображений будут соответствовать стандартам веба.
