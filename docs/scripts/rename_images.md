# [Переименование изображений (rename_images.py)](../../scripts/rename_images.py)

## Назначение
Одноразовый скрипт нормализации имён файлов изображений: lowercase + пробелы→дефисы (приведение к формату слагов).

## Подробное описание
*   `rename_logic(name)` — `name.lower().replace(' ','-')`.
*   `run_rename()` — ищет папку `images` (несколько кандидатов путей) и переименовывает файлы.

## AI-контекст
*   Формат имён должен совпадать с тем, что ожидает [SlugService](../frontend/ground/utils/SlugService.md)/[ItemIconService](../frontend/ground/utils/ItemIconService.md). Утилита разовая (обслуживание ассетов).
