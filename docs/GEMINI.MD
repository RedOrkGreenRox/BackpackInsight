# Правила проекта

## Общие
- Будьте полезными.
- Говорите на русском языке.
- Следуйте существующим в проекте стилям программирования и
  соглашениям (комментарии в коде - не соглашение - его можно игнорировать).

## Python
- Следуйте стандартам PEP 8.
- Используйте подсказки по типу, где это возможно.

## JavaScript
- Используйте современный синтаксис ES6+.
- Предпочитайте "const" и "let", а не `var`.
