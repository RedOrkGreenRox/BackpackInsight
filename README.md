# Backpack Insight 🎒

Веб-приложение для анализа профилей игроков. Загружайте JSON-файлы сохранений, смотрите статистику, инвентарь героев и ищите предметы.

---

## 🚀 Быстрый старт (Docker)

Самый простой способ запустить проект:

1.  **Скопируйте конфиг**:
    ```bash
    cp .env.example .env
    ```

2.  **Запустите**:
    ```bash
    docker-compose up -d --build
    ```

3.  **Готово!**
    *   Сайт: [http://localhost:5080](http://localhost:5080)
    *   API: [http://localhost:8000/docs](http://localhost:8000/docs)

---

## ✨ Что умеет

*   **Витрина**: Красивое отображение статистики и героев из файла сохранения.
*   **Умное сохранение**: Не создает дубликатов и хранит историю из 3-х последних версий профиля.
*   **Поиск**: Мгновенный поиск предметов (даже с опечатками и на других языках).
*   **База знаний**: Полный каталог предметов игры.

---

## 🛠️ Полезные команды

**Запуск тестов:**
```bash
python scripts/run_tests.py
```

**Полный сброс базы данных (удалить всё):**
```bash
python Backend/DB/reset_db.py
```

**Проверка индексов поиска:**
```bash
python scripts/verify_indexes.py
```

---

## 💻 Локальный запуск (без Docker)

1.  Установите зависимости:
    ```bash
    pip install -r Frontend/Web/requirements.txt
    ```
2.  Добавьте папку проекта в `PYTHONPATH`.
3.  Запустите:
    ```bash
    python Frontend/Web/web.py
    ```
